# 代数 algebra (Vaughan Pratt)

_首次发布于 2007 年 5 月 29 日；实质性修订于 2022 年 11 月 3 日_

代数是数学的一个分支，与几何、分析（微积分）、数论、组合学等相似。虽然代数的根源可以追溯到实数和复数等数值领域，但在其完整的普遍性中，它与其他数学领域不同，没有特定的数学领域。几何学处理空间实体，分析处理连续变化，数论处理整数算术，组合学处理离散结构，而代数同样适用于所有这些以及其他数学领域。

_初等代数_已经使用了几个世纪，并在中学教育中教授，它是关于不确定数量或变量 x，y，...的算术。例如，明确的求和 3+4 的结果是明确的数量 7，而不确定的求和 x+y 没有明确的值，但我们仍然可以说它总是等于 y+x，或者当且仅当 x 是-y 或 y+1 时等于 x2-y2。

初等代数提供了有限的方法来处理无限。例如，半径为 r 的圆的面积公式πr2 描述了无限多种可能的计算，每种计算都对应着其变量的可能取值。一个普遍真实的定律表达了无限多种情况，例如单一方程 x+y=y+x 总结了无限多个事实 1+2=2+1，3+7=7+3 等等。方程 2x=4 从无限的可能中选择一个数。而 y=2x+3 用一个有限的方程表达了通过点(0,3)且斜率为 2 的直线上的无限多个点，其解就是这些点。

初等代数通常处理实数或复数值。然而，它的一般方法，尽管不总是具体的操作和定律，同样适用于其他数值域，如自然数、整数、模某个整数 n 的整数、有理数、四元数、高斯整数、p-进制数等等。它们也适用于许多非数值域，如给定集合下的并集和交集运算的子集，给定字母表上的连接和反转运算的单词，给定集合上的组合和逆运算的排列等等。每个这样的_代数结构_，或者简称为_代数_，由其元素的集合和在该域中遵守的运算法则组成，例如整数运算加法 x+y、乘法 xy 和取反−x，或者集合运算并集 X∪Y、交集 X∩Y 和相对于 X 的补集 X′。

法律通常相似但不完全相同。例如，整数乘法在加法上分配，x(y+z)=xy+xz，但反过来不成立，例如 2+(3×5)=17，但(2+3)×(2+5)=35。在将交集视为集合论中乘法的对应物，联合视为加法的对应物的类比中，交集在并集上分配，

X∩(Y∪Z)=(X∩Y)∪(X∩Z)，

至于整数，但与整数不同，联合运算也适用于交集：

X∪(Y∩Z)=(X∪Y)∩(X∪Z)。

而初等代数是在一个固定的代数中进行的，_抽象_或_现代代数_则处理具有某些共同属性的代数类，通常是可表达为方程的类。这个学科在 19 世纪出现，传统上是通过群、环和域的类别来引入的。例如，任何在加法和减法运算下的数系统形成一个阿贝尔（交换）群；然后通过引入乘法来转向环，再进一步转向域与除法。常见的四则运算计算器提供了实数域的四个运算。

群的抽象概念完全是根据一组数字而不是数字集合来定义的，而是作为一个任意集合，配备了一个二元运算 xy，一个该运算的一元逆元 x-1，以及满足群特征方程的单位元 e。群与日常初等代数中没有遇到的一个引人注目的新颖之处是它们的乘法不一定是交换的：xy 和 yx 可以不同！例如，三个事物的六种可能排列的群 S3 不是交换的，可以通过交换字母对_dan_中的相邻字母来看出。如果你先交换左边的两个字母，然后再交换右边的两个字母，你得到_adn_，然后是_and_，但是如果你按照相反的顺序进行这些交换，你得到_dna_，然后是_nda_，而不是_and_。同样，魔方的 43,252,003,274,489,856,000 次操作的群和球体的旋转的无限群 SO(3)也不是交换的，尽管圆的旋转的无限群 SO(2)是交换的。四元数乘法和矩阵乘法也是非交换的。阿贝尔群通常被称为加法群，它们的群运算被称为加法 x+y，而不是乘法 xy。

群、环和域只是抽象代数的表面。向量空间和更一般的模是环的受限形式，其中乘法的操作数要求是一个标量和一个向量。幺半群通过去除逆元来推广群；例如自然数形成一个幺半群，但由于缺乏取反操作，不是一个群。布尔代数抽象了集合的代数。格推广了布尔代数，去除了补和分配律。

许多数学分支发现代数是一种有效的工具，因此它们衍生出了代数的子分支。代数逻辑、代数数论和代数拓扑都受到了广泛的研究，而代数几何和代数组合学则有专门的期刊。

代数出于至少两个原因引起了哲学的兴趣。从数学基础的角度来看，代数在其领域独立性和与形式逻辑的密切关联方面与其他数学分支截然不同。此外，初等代数和抽象代数之间的二分法反映了一种推理的二元性，笛卡尔二元论的发明者笛卡尔会对此赞赏，其中前者处理推理过程，后者处理推理对象，分别对应于数学的心智和身体。

代数还在澄清和突出逻辑概念方面发挥了重要作用，这是准确哲学核心的基础。从亚里士多德的三段论逻辑向更代数形式的逻辑迈出的第一步是由布尔在 1847 年的小册子中采取的，随后在 1854 年的更详细的论著《思维的法则》中进一步发展。初等代数和现代代数之间的二分法随后开始在逻辑的发展中出现，逻辑学家在形式主义方法（如弗雷格、皮亚诺和罗素所倡导的方法）和代数方法（如皮尔斯、施罗德和塔斯基所采用的方法）之间存在着强烈的分歧。

## 1. 初等代数

初等代数处理数字项，即常数 0、1、1.5、π、变量 x、y 等，以及使用+、-、×、÷、√等运算符组合而成的项，如 x+1、x×y（标准缩写为 xy）、x+3y 和√x。

这些项可以单独用于公式中，例如πr2，也可以用于作为等式的法则，如 x+y=y+x，或作为约束条件，如 2x2−x+3=5x+1 或 x2+y2=1。

定律总是正确的；虽然它们与约束的形式相同，但它们只是空洞地约束，因为它们的变量的每个赋值都是一个解。约束条件 x2+y2=1 有一个连续的解集，形成一个_形状_，在这种情况下是一个半径为 1 的圆。约束条件 2x2−x+3=5x−1 有两个解，x=1 或 2，并且可能在解决_文字问题_或确定两条曲线的交点时遇到，例如抛物线 y=2x2−x+3 和直线 y=5x−1 的交点。

### 1.1 公式

公式是手工或机器计算值时使用的术语。虽然一些物体的属性可以直接测量，例如长度和质量，但其他属性，如面积、体积和密度，则需要根据更容易观察到的值，借助适当的公式进行计算。例如，一个长为 L 英寸，宽为 W 英寸的矩形的面积用平方英寸为单位的公式为 LW，半径为 r 的球的体积为 4πr3/3，质量为 M，体积为 V 的固体的密度由 M/V 给出。

公式可以组合成更多的公式。例如，通过将上述球体体积的公式代入上述固体密度的公式中的 V，可以得到质量为 M，半径为 r 的球体的密度。得到的公式 M/(4πr3/3)就是所需的密度公式。

### 1.2 法则

法则或等式是对其变量的所有适用值都成立的方程。例如，交换律法则

x+y=y+x

对于所有的实数 x 和 y 都成立。同样，结合律

x+(y+z)=(x+y)+z

对于所有实数 x、y 和 z 都成立。另一方面，当 x/(y/z)=zx/y 对于所有数值 x 成立时，为了避免除以零的非法操作，它只对非零的 y 和 z 成立。

当一个法则适用于其变量的所有数值时，它也适用于这些变量的所有表达式值。将 x=M，y=4πr3，和 z=3 代入上一段的最后一个法则得到 M/(4πr3/3)=3M/(4πr3)。左边是我们在前一节中介绍的密度公式，根据上述法则的这个实例，可以得出右边是一个等价的密度公式，因为它给出与左边相同的答案。这个新的密度公式用乘法取代了两个除法之一。

### 1.3 词问题

如果 Xavier 在四年后的年龄是他现在的三倍，那么他现在多大？我们可以通过代数来解决这个“文字问题”，将其形式化为方程式 3x=x+4，其中 x 是 Xavier 现在的年龄。左边表示 Xavier 现在的年龄的三倍，而右边表示他四年后的年龄。

解决这类方程的一般规则是，对方程的任何解也是通过对两边应用某种操作得到的方程的解。在这种情况下，我们可以通过从两边减去 x 来简化方程，得到 2x=4，然后将两边都除以 2，得到 x=2。所以 Xavier 现在两岁。

如果 Xavier 的年龄是 Yvonne 的两倍，并且是她年龄的平方的一半，那么他们各自多大？这个问题比之前的例子更复杂，有三个方面：它有更多的未知数，更多的方程，以及更高次的项。我们可以用 x 表示 Xavier 的年龄，用 y 表示 Yvonne 的年龄。这两个限制条件可以形式化为方程 x=2y 和 x=y2/2，后者是二次方程。

由于两个等式的右边都等于 x，我们可以推断出 2y=y2/2。诱人的是将两边都除以 y，但如果 y=0 呢？实际上，y=0 是一个解，对应于 Xavier 和 Yvonne 都是新生儿的情况，此时 x=2y=0。将这个解放在一边，我们现在可以寻找 y 不为零的解，将两边都除以 y。这样得到 y=4，此时 x=2y=8。所以现在我们有了第二个解，其中 Xavier 八岁，Yvonne 四岁。

在没有其他信息的情况下，这两个解决方案都是合法的。如果问题进一步指明 Yvonne 是一个幼儿，或者 Xavier 比 Yvonne 年长，我们可以排除第一个解决方案。

### 1.4 笛卡尔几何学

平面上的直线、圆和其他曲线可以用代数方式表达，这种方式被称为笛卡尔坐标系，以其发明者勒内·笛卡尔命名。这些坐标系是相对于平面上的一个特殊点，称为原点 O 来定义的。每个点都可以用一对数字来表示，表示它距离原点 O 向右和向上的距离。例如，对于一对数字(2.1, 3.56)，它指定了一个点，该点距离原点 O 向右 2.1 个单位，水平测量，向上 3.56 个单位，垂直测量；我们称 2.1 为该点的 x 坐标，3.56 为该点的 y 坐标。任何一个坐标都可以是负数：对于一对坐标(-5, -1)，它对应于一个点，该点距离原点 O 向左 5 个单位，向下 1 个单位。原点 O 本身的坐标是(0, 0)。

_直线。_ 给定一个关于变量 x 和 y 的方程，例如(2, 7)这样的点被称为该方程的一个解，当将 x 设为 2，y 设为 7 时，方程成立。例如，方程 y=3x+5 的解包括点(0, 5)，(1, 8)，(2, 11)，等等。其他解包括(.5, 6.5)，(1.5, 9.5)，等等。所有解的集合构成了通过点(0, 5)和(1, 8)的唯一直线。我们将 y=3x+5 称为该直线的方程。

_圆形_。根据毕达哥拉斯定理，两点（x，y）和（x'，y'）之间的距离的平方由（x'−x）2+(y'−y）2 给出。作为特例，点（x，y）到原点的距离的平方是 x2+y2。由此可知，距离原点 r 的点是方程 x2+y2=r2 中 x 和 y 的解。但这些点恰好是以 O 为中心的半径为 r 的圆形。我们将这个方程与这个圆形等同起来。

_变量_。任何关于 x 和 y 的多项式的根形成了平面上的一条曲线，称为多项式的_一维变量_，其次数为多项式的次数。因此，直线的次数为 1，表示为多项式 ax+by+c，而以（x'，y'）为中心的圆的次数为 2，表示为多项式（x−x'）2+(y−y'）2−r2。有些变量可能不包含任何点，例如 x2+y2+1，而其他变量可能包含一个点，例如 x2+y2，其原点为其一个根。然而，一般情况下，二维变量将是一条曲线。这样的曲线可能会交叉，或者有一个尖点，甚至分裂成两个或更多不相连的部分。

_空间_ 通过添加第三个变量 z 来将二维平面推广为三维空间，对应于第三个维度。传统的方向是第一维从西到东，第二维从南到北，第三维从下到上。点则是三元组，例如点(2,5,−3)是距离原点向东 2 个单位，向北 5 个单位，向下 3 个单位。

_平面和球体_。这些是空间中线和平面中圆的对应物。例如方程 z=3x+2y 定义的不是一条直线，而是一个平面，这种情况下是通过点(0, 1, 2)，(1, 0, 3)和(1, 1, 5)的唯一平面。以原点为中心、半径为 r 的球体由方程 x2+y2+z2=r2 给出。多项式在 x、y 和 z 中的根构成了空间中的一个二维变量，其次数与多项式的次数相同，就像一维变量一样。因此，平面的次数为 1，球体的次数为 2。

这些方法通过添加更多的变量可以推广到更高的维度。虽然我们在物理上所经历的几何空间仅限于三维，但在概念上，抽象数学空间的维度数量是没有限制的。就像一条线是二维平面的一维子空间，一个平面是三维空间的二维子空间一样，每个都可以用一个方程来确定，一个_超平面_是四维空间的三维子空间，也可以用类似 w=2x−7y+z 的方程来确定。

## 2. 抽象代数

初等代数固定了某个域，通常是实数或复数，并在该域内处理满足方程的问题。_抽象_或_现代代数_通过固定一些方程集合 A，并研究那些使这些方程成为恒等式的域来颠倒这个观点。例如，如果我们取由加法、减法、乘法和常数 0 和 1 表示的所有恒等式的集合，并且这些恒等式对整数成立，那么这些方程恒等成立的代数结构正好是带单位元的交换环。

从历史上看，现代代数一词来自于范德瓦尔登的经典著作的前三版的标题，第四版于 1955 年改名为简单的“代数”。第一卷讨论了群、环、一般域、向量空间、良序集和实数域，而第二卷主要考虑了线性代数、代数（作为具有兼容乘法的向量空间）、表示论、理想论、整数代数元、代数函数和拓扑代数。一方面，现代代数已经远远超出了这个课程，另一方面，这一大量的材料已经超出了数学博士研究生所能掌握的常识范围，因为他们的典型课程时间太短，无法在专注于自己的专业领域的同时掌握所有这些材料。

抽象代数的一个核心特征是存在着一些领域，其中熟悉的法则不再适用。一个引人注目的例子是乘法的交换性，在引言中我们注意到，即使是像三个字母的六个排列这样简单的群的乘法也不一定满足交换律。

### 2.1 半群

我们从集合 X 上的二元运算的概念开始，即一个函数 f:X2→X，使得对于 X 中的所有元素 x 和 y，f(x,y)是 X 的一个元素。当这样的运算满足对于 X 中的所有 x、y 和 z，f(f(x,y),z)=f(x,f(y,z))时，称其为_结合_的。

一个_半群_是一个带有结合运算的集合，称为半群的_乘法_，用 xy 表示，而不是 f(x,y)。

一个元素与自身的乘积被表示为 x 的平方，类似地，xxx 被表示为 x 的立方，依此类推。

_例子_

* 给定字母表上所有非空单词的集合，通过连接操作得到。
* 在集合 X 上，通过函数复合操作得到的所有函数 f:X→X 的集合。
* 固定正整数 n 下的 n×n 整数矩阵的集合，通过矩阵乘法运算。

单词 u 和 v 的连接 uv 是可结合的，因为当一个单词被切成两部分时，无论切割点在哪里，两部分的连接都是原始单词。单词**al**和**gebra**的连接与**algeb**和**ra**的连接相同，说明连接运算对于情况 x=**al**，y=**geb**，z=**ra**是可结合的。

两个函数 f 和 g 的复合函数 f⋅g 通过以下推理是可结合的：

(f⋅(g⋅h))(x)=f((g⋅h)(x))=f(g(h(x)))=(f⋅g)(h(x))=((f⋅g)⋅h)(x)

对于集合 X 中的所有 x，有 f⋅(g⋅h)=(f⋅g)⋅h。

当 H 是 G 的子半群时，H 是 G 的_子半群_。当 H 是 G 的子集且 G 的乘法在 H 上的限制与 H 的乘法相同时，H 是 G 的子半群。换句话说，G 的子半群是 G 的子集 H，对于 H 中的所有 x 和 y，xy 也在 H 中。

_例子_

* 给定字母表上的所有非空单词的半群具有偶数长度的单词作为子半群；然而奇数长度的单词不构成子半群，因为两个奇数长度单词的连接不是奇数长度。
* 在函数复合下，集合 X 上的所有函数 f:X→X 的半群具有子半群：单射函数、满射函数和双射函数。

当对于集合 X 中的所有 x 和 y，二元运算满足 f(x,y)=f(y,x)时，称其为_可交换_的二元运算。一个_可交换半群_是一个具有可交换运算的半群。到目前为止，所有的例子都是非可交换半群。以下是可交换情况的示例。

_示例_

* 正整数集合在加法下。
* 所有整数的集合，使用加法运算。
* 在一个字母字母表下的单词集合，使用连接运算。
* 任何 AND、OR、XOR 操作下的位（二进制数字）集合{0,1}。

%%

* 平面上向量加法下的右上象限的向量集合。
* 相同，但省略原点。
* 所有三维向量的集合，通过向量加法运算。
* 在整数系数下，一元多项式 x 的集合，通过多项式加法运算。
* 对于固定的正整数 m 和 n，矩阵加法下的 m×n 整数矩阵集合。

对于 f，如果对于 X 中的所有 y，f(x,y)=y，则 X 中的元素 x 是 f 的_左单位元_；如果对于 X 中的所有 y，f(y,x)=y，则 X 中的元素 x 是 f 的_右单位元_。对于 f 来说，既是左单位元又是右单位元的元素是_f 的单位元_。操作 f 只能有一个单位元，因为当 x 和 y 都是单位元时，它们都等于 f(x,y)。

一个_幺半群_是一个包含幺元的半群，用 1 表示。

_例子_

* 连接的单位元是空字。因此，在允许空字的情况下，连接的字形成一个幺半群。
* 加法的单位元是零，或者在向量加法的情况下是原点。因此，如果上述任何一个半群的操作是加法，并且只有包含零时，它才形成一个幺半群。
* 组合的单位元是定义为 1X:X→X 的单位函数，对于 X 中的所有 x，有 1X(x)=x，因此在集合 X 上的所有函数的半群形成一个幺半群。

当一个幺半群 H 是幺半群 G 的_子幺半群_时，它是 G 的子半群，包括 G 的单位元。

### 2.2 群

当一个幺半群的两个元素 x 和 y 满足 xy=1 时，我们称 x 是 y 的左逆元，y 是 x 的右逆元。一个既是 x 的左逆元又是右逆元的元素 y 被称为 x 的逆元。

一个_群_是一个每个元素都有逆元的幺半群。

群的基数通常被称为它的_阶_。当存在最小的正整数 n 使得 gn=1 时，群元素 g 被称为阶为 n 的元素。

_示例_

* 整数的幺半群是指加法下的整数集合，因为每个整数 x 都有逆元−x。
* 在 XOR 运算下的二进制位（比特）集合{0,1}，因为每个比特都是自己的逆：0XOR0=1XOR1=0。这只是整数模 n（例如 3+4=2（mod 5））的幺半群{0,1,2,...,n-1}中 n=2 的情况。这里 0 是自己的逆，非零数 m 的逆是 n-m。
* 在组合下的双射或置换 f:X→X 的幺半群 SX，因为每个置换都有逆 f-1。当 X 有 n 个元素时，Sn 的阶为 n!。当且仅当 n≤2 时，Sn 是可交换的。
* 平面上关于一个点的旋转的幺半群在合成下封闭，因为每个旋转都可以逆转。这个群被称为圆群，表示为 SO(2)。
* 三维空间中关于一个点的旋转的幺半群在合成下封闭，同样因为每个旋转都可以逆转。这个群表示为 SO(3)。
* 对称群（旋转和反射）是正 n 边形绕其中心将 n 边形带回自身的幺半群，具有可逆性。这个群被称为_dihedral group_ Dn，阶数为 2n。与 Sn 类似，当且仅当 n≤2 时，Dn 是可交换的；特别地，D3=S3。

一个群 G 的_子群_是 G 的子幺半群，且在逆元下封闭。自然数和偶数的幺半群都是整数幺半群（加法下）的子幺半群，但只有后者是子群，因为它在取反下封闭，而自然数不是。

一个_阿贝尔群_是一个其运算是可交换的群。阿贝尔群的群运算通常被称为加法而不是乘法，有时也被称为加法群。

一个_循环群_是一个具有元素 g 的群 G，使得 G 的每个元素都可以表示为 gi 的形式，其中 i 是一个正整数。循环群是可交换的，因为 gigj=gi+j=gjgi。整数加法群以及任何正整数 n 的模 n 整数群都是循环群，其中 1 在每种情况下都是一个生成元。所有循环群都同构于其中之一。当群的阶数大于等于 3 时，总是存在其他的生成元，例如-1，而对于素数阶的群，每个非零元素都是一个生成元。

### 2.3 环

一个_环_是一个阿贝尔群，通过具有第二个操作的幺半群，即环的_乘法_。零_使其为零元_，即 0x=x0=0。此外，乘法在加法（群操作）中对两个参数都满足分配律。也就是说，x(y+z)=xy+xz 和(x+y)z=xz+yz。

_例子_

* 整数的加法群，通过整数乘法扩展。
* 用整数系数扩展的一元多项式的加法群，通过多项式乘法进行扩展。
* 用矩阵乘法进行扩展的固定正整数_n_的整数_n×n_矩阵的加法群。
* 数字形式为 a+b√2 的一组数，其中 a 和 b 是整数，因为(a+b√2)(c+d√2)=ac+2bd+(bc+ad)√2。
* 整数模 n 的一组数，其中 n≥2，因为(x+an)(y+bn)=xy+(xb+ya+abn)n。

在除了最后一个例子之外，整数（除了给出矩阵大小的整数 n）可以被有理数、实数或复数替代。当用实数替换整数时，第四个例子变成了实数环，因为即使 b 为零，a 可以是任意实数。然而，当用有理数替换时，环包括有理数，但不仅限于此，因为√2 是无理数，但不包含例如√3。

### 2.4 域

一个_域_是一个环，其中非零环元素的乘法幺半群是一个阿贝尔群。也就是说，乘法必须是可交换的，并且每个非零元素 x 必须有一个倒数 1/x。

_例子_

* 有理数环，因为有理数的乘法是可交换的，而且每个非零有理数 m/n 都有倒数 n/m。
* 实数环和复数环，出于类似的原因。
* 数字形式为 a+b√2 的数环，其中 a 和 b 是有理数，因为只有当 a=b=0 时，a+b√2 才为零，否则它的倒数为(a−b√2)/(a2−2b2)；被称为“二次无理数”的域。
* 模素数 p 的整数环 Zp，因为非零数的乘法幺半群包含一个数 g，使得 gp−1=1，并且每个非零数都可以写成 gi 的形式，其中 i 是某个整数，因此它有一个逆元，即 gp−1−i。

最后一个例子不能直接推广到其他模数。然而，对于任何一个是素数的幂 pn 的模数，可以证明存在一种唯一的乘法，使得群 Zpn 成为一个环，使得环中的非零元素在乘法下成为一个循环（因此是可交换的）群，从而使得环成为一个域。以这种方式构造的域是唯一的有限域。

### 2.5 应用

为什么要学习整个类别？嗯，以整数集合 Z 为例，加上加法 x+y、取反-x 的一元运算和常数 0 的二元运算。这些运算和常数满足各种定律，如 x+(y+z)=(x+y)+z，x+y=y+x，x+0=x 和 x+(−x)=0。现在考虑任何其他代数，其运算不仅具有相同的名称，而且满足相同的定律（可能还有更多），称为这些定律的“模型”。这样的代数可以用于以下任何目的。

(i) 它可以告诉我们整数的等式定律在多大程度上表征了整数。由于模 2 下的整数集合{0,1}在加法和取反下满足整数的所有定律，我们立即可以看出，没有任何一个整数的等式属性告诉我们整数有无限多个。另一方面，整数的等式理论的任何有限模型都必然满足一些整数不满足的定律，特别是左边有 x 的数量为模型的大小的定律 x+x+…+x=0。由于整数的等式理论不包含这样的定律，我们可以从整体上看出，整数必须是一个无限集合。另一方面，有理数在加法和取反下与整数完全满足相同的等式属性，因此这个理论不能以足够精确的方式表征整数的加法和减法代数，以将其与有理数区分开来。

(ii) 它可以为我们提供一个有用的新领域，可以替代整数在任何仅依赖于整数等式性质的应用中，但在其他（必然非等式的）有用方面与整数不同。例如，有理数满足我们刚才提到的相同法则，但不同之处在于它具有“密度”属性，即在任意两个有理数之间存在另一个有理数。另一个区别是它支持除法：而两个整数的比率通常不是整数，两个有理数的比率总是有理数。实数也满足相同的方程，与有理数一样是密集的并且支持除法。然而，与有理数不同的是，实数具有“完备性”属性，即任何非空实数集的所有上界集合要么为空，要么具有最小成员，这对于收敛序列有极限是必要的。

这个想法可以扩展到其他运算，比如乘法和除法，就像域一样。在笛卡尔几何中，使用复数的一般化特例特别有用。当 x 和 y 在实数域上变化时，x2+y2=1 描述了二维欧几里德圆，但当变量在复数上变化时，这个方程描述了圆的复数对应物，可以想象为嵌入在四个实维度中的二维表面（将复平面视为具有两个实维度）。或者，如果变量在模 7 的整数范围内变化，这些整数在通常的算术运算模 7 下形成一个域，那么圆由八个点组成，即(±1,0),(0,±1)和(±2,±2)。关于欧几里德圆的某些定理在纯代数上仍然可证明，因为证明所依赖的所有方程在这些其他域中仍然成立，例如直线与圆最多相交于两点的定理。

(iii) 它可以帮助我们决定一些旨在公理化整数的等式法则列表是否“完备”，即整数满足的任何等式是否都可以从该列表中的法则推导出来。如果某个结构满足列表中的所有公理，但不满足整数满足的某个其他等式，那么我们就有了一个证明该公理化不完备的证据。另一方面，如果我们可以展示如何从整数的代数中仅限于某些代数构造来构建满足公理的任何代数，那么根据 Birkhoff 的一个适用于这些构造的定理，我们可以推断该公理化是完备的。

(iv) 它可以提供除列举公理的标准方式之外的另一种定义类的方法。在这个案例中，具有常数、一元操作和二元操作，并满足整数满足的所有法则的所有代数的类，恰好是阿贝尔群的类。

## 3. 代数学

代数学是抽象代数之后的下一个抽象层次。而初等代数处理特定代数（如实数域或复数域）中的方程推理，抽象代数研究特定类别的代数（如群、环或域），而代数学则研究代数的类别的类别。就像抽象代数将群、环和域作为其基本类别一样，代数学也将变种、拟变种和初等类别作为其基本类别的类别。

一个理论的模型是一个结构，其中该理论的所有方程都是恒等式。术语是由变量和常数使用理论的运算构建起来的。方程是一对术语；当两个术语在对术语中出现的 n 个变量的所有赋值下相等时，它被代数满足，或者当它们表示相同的 n 元运算时。拟方程是由一组方程（称为前提或前件）和另一个方程（结论）组成的一对；当满足前提的术语中出现的 n 个变量的所有赋值下，结论的两个术语相等时，它被代数满足。一阶公式是关系术语的量化布尔组合。

一个变种是一组方程的所有模型的类。一个拟变种是一组拟方程的所有模型的类。一个初等类是一组一阶公式的所有模型的类。

准变种比变种或基本类别受到的关注要少得多，因此我们在这里对它们的介绍很少。基本类别在这本百科全书的其他地方已经有足够深入的讨论，所以我们不需要在这里考虑它们。因此，我们在本节中专注于变种。

阿贝尔群、群、环以及在给定域上的向量空间都构成了变种。

这个领域的一个核心结果是一个格子是某个代数的子代数格子当且仅当它是某个代数的同余格子。这种类型的格子被称为_代数格子_。当一个代数的同余关系可以互换时，它的同余格子是模块化的，这是一个有利于特定有限代数分析的强条件。

### 3.1 概念

代数中的常见数论定理以代数形式出现。当一个代数 A 的同余格是由 A 和一个元素的代数组成的二元格时，称其为直接不可约或简单的，这与素数 p 的约数格只有 p 和 1 两个元素的概念相对应。然而，算术基本定理的对应物，即每个正整数都可以唯一地分解为素数的乘积，需要比直接乘积更精细的乘积。Birkhoff 的子直积概念使他能够证明子直积表示定理，即每个代数都可以作为其子直积不可约商的子直积。虽然有许多子直积不可约群，但唯一的子直积不可约布尔代数是初始的或二元的，而满足 xn=x（其中 n>1）的子直积不可约环正好是有限域。

另一个核心主题是对偶性：布尔代数对偶于 Stone 空间，完全原子布尔代数对偶于集合，带有顶和底的分配格对偶于偏序集，代数格对偶于半格，等等。对偶性提供了两种观察代数的方式，其中一种可能在应用中更具洞察力或更易于处理。

变种的结构作为某个等式理论的所有模型的类也非常有趣。这个领域最早的结果是 Birkhoff 的定理，即一个代数类是一个变种，当且仅当它在商（同态像）、子代数和任意（包括空和无限）直积的形成下是封闭的。这个“现代代数”结果构成了等式逻辑的模型的完备性定理。它的基本对应物是关于自由代数 F(V)上的等式理论的定理，F(V)被定义为使用来自 V 的变量的推导闭合的等式集，这些等式集恰好是它的替代等价关系。

局部有限的变种是其有限生成的自由代数是有限的，例如指向集合、图（无论是有向还是无向的）和分配格。可置换等价关系的变种是其所有代数都是可置换等价关系的。Maltsev 通过对它们的理论提出了一个必要且充分的条件来刻画这些变种，即 F(3)包含一个操作 t(x,y,z)，使得 t(x,x,y)=t(y,x,x)=y 在该理论中成立。类似的概念还有等价关系分配性和等价关系模块性，对于具有这些性质的代数变种存在类似的句法刻画。这个领域最近发展起来的一个有力工具是 McKenzie 提出的温顺等价关系的概念，有助于研究有限代数的结构。

在代数学派中，已经定义了一种理解签名操作形成一个集合的变量。范畴论的洞察力，特别是将变量定义为范畴 C（在普通普遍代数中为集合）的自函子范畴 CC 中的幺半群对象的表达方式，表明当操作可以形成一个适当的类时，可以获得更清晰和更一般的变量概念。例如，完全半格（CSLat）和完全原子布尔代数（CABA）这两个重要的类只有在这个更广泛的签名概念下才形成变量。在狭义的代数意义上，一个变量的对偶永远不可能是一个变量，而在更广泛的幺半群变量概念中，集合的变量 Set 是 CABA 的对偶，而 CSLat 是自对偶的。

### 3.2 等式逻辑

_公理系统。_ 恒等式也可以用来将方程转化为等价方程。当这些方程本身是某个域的恒等式时，它们转化后的方程仍然是该域的恒等式。因此，我们可以从一些有限的恒等式出发，制造出无限多个新的恒等式。

例如，如果我们从只有两个恒等式 (x+y)+z=x+(y+z) 和 x+y=y+x 出发，通过以下一系列的转化，我们可以得到恒等式 (w+x)+(y+z)=(w+y)+(x+z)。

（w+x）+（y+z）=（（w+x）+y）+z=（w+（x+y））+z=（w+（y+x））+z=（（w+y）+x）+z=（w+y）+（x+z）

从旧的身份制造新的身份的过程称为_推导_。 从给定的身份集 A 开始通过推导可以生成的任何身份称为 A 的_结果_。 A 的所有结果的集合称为 A 的_推导闭包_。 我们将 A 称为其推导闭包的_公理化_。 如果一个集合是其自身的推导闭包，则称为_推导闭合_。 可以直接证明_一个集合是推导闭合的当且仅当它是某个集合的推导闭包_。

一个_等式理论_是一个推理封闭的等式集合，等价于某个等式集合 A 的所有推论的集合。每个理论总是将自身作为自己的公理化，但通常也会有更小的公理化。具有有限公理化的理论被称为_有限基础_或_有限公理化_。

_**有效性。**_ 有限基础的理论可以被有效地枚举。也就是说，给定一个有限的等式集合 A，可以编写一个计算机程序，以这样的方式打印 A 的推论，使得 A 的每个推论都会在无限列表的某个有限位置出现。当我们将 A 的要求从有限放宽到仅仅是可以有效地枚举时，得出的结论是相同的。也就是说，如果公理化可以有效地枚举，那么它的推导闭包也可以有效地枚举。

（在调和有限与无限的过程中，请记住，如果我们按顺序列出所有自然数 0、1、2、…，我们得到一个无限列表，其中每个成员与起始点之间的距离有限，而且每个成员都有一个明确定义的前驱（除了 0）和后继。只有当我们试图在“末尾”用无限的数字填充这个列表时，这个原则才会失效。

一种将“末尾”视为可能有更多元素的方式是考虑形如 1/n 的有理数，其中 n 是非零整数，按递增顺序排列。这个列表开始是-1/1，-1/2，-1/3，...，在列出了无数个这种形式的负有理数后，没有最大值，然后切换到正有理数，没有第一个这样的数，最后以 1/3，1/2，1/1 结束。整个列表是离散的，因为在这个有理数子集中，除了端点-1/1 和 1/1 之外的每个有理数都有一个明确定义的前驱和后继，这与所有在-1/1 和 1/1 之间的有理数的情况不同。如果我们在“中间”引入有理数 0，它既没有前驱也没有后继。）

_**代数逻辑。**_ 我们对推理的非正式描述可以用五个规则来形式化，用于从旧的等式中产生新的等式。在下面的规则中，s 和 t 表示任意项。

（_R_1）

从无中推断 t=t。

（_R_2）

从 s=t 推导出 t=s。

（_R_3）

从 s=t 和 t=u 推断出 s=u。

（_R_4）

从 s1=t1,s2=t2,…,sn=tn 推导出 f(s1,s2,…,sn)=f(t1,t2,…,tn)，其中 f 是一个 n 元操作。

(_R_5)

从 s=t 推导出 s′=t′，其中 s′和 t′是通过在 s 和 t 中一致地替换变量的项得到的。

在这个上下文中，“一致地”意味着如果一个项被替换为给定变量的一个出现，那么同样的项必须被替换为 s 和 t 中该变量的所有出现。例如，我们不能仅仅依靠_R_5 来证明在 x+y=y+x 的左边将 u+v 替换为 x，在右边将 v+u 替换为 x，尽管其他规则可能允许这样做。

一个等式理论作为一组项对的集合，相当于对所有项的二元关系。规则_R_1-_R_3 分别对应于这个二元关系的自反性、对称性和传递性，即这三个规则断言等式理论是一个等价关系。规则_R_4 表达了这个二元关系是一个同余关系的进一步属性。规则_R_5 进一步断言这个关系是一个可替代的同余关系。可以证明，如果一个二元关系在项的集合上是一个等式理论，那么它也是一个可替代的同余关系。因此，这五个规则完全公理化了等式逻辑，即通过有限次应用这五个规则可以产生出一组方程的所有推论。

一种变量在定义上是某个等式理论的模型类。1935 年，Birkhoff 提供了一种等价的变量描述，即任何在商（同态像）、直积和子代数下封闭的类。这些概念的定义如下。

给定两个代数（X，f1，...，fk）和（Y，g1，...，gk），一个_同态_ h：（X，f1，...，fk）→（Y，g1，...，gk）是一个函数 h：X→Y，满足对于每个 i 从 1 到 k，都有 h（fi（x0，...，xni-1））= gi（h（x0），...，h（xni-1）），其中 ni 是 fi 和 gi 的元数。

一个代数的_子代数_是代数中的元素集合，该集合在代数的运算下封闭。

设 I 是一个任意的集合，可以是空集、有限集或无限集。一个由索引集 I 索引的代数_族_ ⟨Ai⟩i∈I 由每个元素 i 对应的代数 (Ai,fi1,…,fik) 组成。我们定义这样一个族的_直积_ ΠAi （或完整写作 Πi∈IAi）如下。

ΠAi 的底层集合是底层集合 Xi 的笛卡尔积ΠXi，由那些第 i 个元素是 Xi 的元素的 I 元组组成。（I 甚至可以是不可数的，但在这种情况下，ΠXi 的非空性作为ΠXi 的非空性的结果等同于选择公理。这一点对 Birkhoff 定理的任何建设性应用都应该牢记。）

ΠAi 的第 j 个操作，具有 arity nj，接受ΠXi 的元素的 nj 元组 t，并产生 I 元组⟨fij(ti1,…tinj)⟩i∈I，其中 tik 是 t 的第 k 个分量的第 i 个分量，k 从 1 到 nj。

给定两个代数 A、B 和一个同态 h:A→B，_同态像_ h(A) 是由形如 h(a) 的元素组成的 B 的子代数，其中 a 是 A 中的元素。

给定一个代数类 C，我们用 P(C)表示由 C 的代数族的直积形成的所有代数的类，用 S(C)表示 C 的代数的所有子代数的类，用 H(C)表示 C 的代数的所有同态像的类。

相对而言，很容易证明任何由 C 的所有成员满足的方程也由 P(C)、S(C)和 H(C)的所有成员满足。因此，对于一个变种 V，P(V)=S(V)=H(V)。

Birkhoff 定理是其逆命题：对于任何满足 P(C)=S(C)=H(C)的类 C，C 是一个变种。实际上，该定理稍微更强：对于任何类 C，HSP(C)是一个变种。也就是说，为了构建 C 的所有模型的理论，只需首先在直积下封闭 C，然后在子代数下封闭 C，最后在同态像下封闭 C；也就是说，后续的封闭不会损害先前的封闭，前提是按照 P、S 和 H 的顺序进行操作。

Birkhoff 定理的一个基本应用是证明一个类 C 的一个提议的公理化的完备性。给定一个任意的公理模型，只需证明该模型可以构造为 C 类代数的直积的子代数的同态像。

这种完备性技术与前一节中对等式逻辑规则的完备性相辅相成。

## 4. 代数

### 4.1 向量空间

兄弟姐妹群、环和域的类别是在任何给定域上的向量空间，构成了线性代数的宇宙。向量空间适用于两种相反的方法：公理化或抽象化，以及合成或具体化。公理化方法将域（由此得到环，由此得到群）作为先决条件；它首先定义了一个 R-模的概念，作为一个给定环 R 上的可交换群与标量乘法，然后定义了一个向量空间为一个 R-模，其中 R 是一个域。合成方法通过将实数上的向量空间表示为实数的 n 元组，以及从 m 维向量空间到 n 维向量空间的线性变换表示为实数的 m×n 矩阵来进行。对于包括无限维的向量空间的完全普遍性，n 不需要限制为有限数，而可以是任意基数。

抽象方法，如 Mac Lane 和 Birkhoff 等经典文本所采用的方法，具有一定的纯粹主义吸引力，非常适合数学专业的学生。具体方法的好处是可以将微积分或其他较少的内容替代为群-环-域作为先决条件，适用于科学家和工程师的服务课程，只需要有限维矩阵代数，这在实际应用中具有巨大的适用性。在编码理论、量子计算等领域中，使用其他域上的线性代数，特别是有限域，抽象方法往往更适合。

对于任何域 F，对于给定的有限维度，存在且仅存在一个 F 上的向量空间，这是抽象方法中的一个定理，但是在具体方法中的表示中是一个直接的结果（该定理用于关联这两种方法）。

具体方法的另一个直接结果是 F 上有限维向量空间的对偶性。对于任何维度的向量空间 V，对应着它的对偶空间 V∗，由 V 上的_泛函_组成，泛函定义为线性变换 f:V→F，将域 F 视为一维向量空间。这些泛函在坐标上的加法（f+g)(u)=f(u)+g(u)）和乘法（(xf)(u)=x(f(u))）下构成一个向量空间，其中 x 是 F 中的任意标量，我们将 V∗定义为这个空间。向量空间上的这种运算扩展到线性变换 f:U→V 上，定义为 f∗:V∗→U∗，使得 f 将每个泛函 g:V→F 映射到 g⋅f:U→F。重复这个运算会产生一个向量空间，在有限维情况下，它与 V 同构，即 V≅V∗∗，使得这个运算成为一个对合运算。有限维向量空间对偶性的本质在于它的对合性质以及线性变换的反转。

这种二元性在具体方法中很容易通过将从 U 到 V 的线性变换视为 m×n 矩阵来进行可视化。这种二元性简单地转置矩阵，同时保持矩阵乘法本身的机制不变。很明显，这个操作是一个反射，它将线性变换一个 n 维空间 U 到一个 m 维空间 V 的 m×n 矩阵转置为一个线性变换一个 m 维空间 V∗到一个 n 维空间 U∗的 n×m 矩阵。

### 4.2 关联代数

在向量空间 V 上的线性变换 f：V→V 可以进行加法、减法和标量乘法，每种情况下都是逐点进行的，因此形成一个向量空间。当空间具有有限维度 n 时，线性变换可以表示为 n×n 矩阵。

此外，它们可以进行组合，从而形成一个带有双线性结合操作的向量空间，即组合。在有限维情况下，组合就是通常的矩阵乘积。具有这种乘积的向量空间构成了_结合代数_。在同构的情况下，所有的结合代数都是以这种方式出现的，无论是有限维还是无限维，这为概念提供了一个令人满意和有洞察力的特征描述，而不是在此处给出的公理描述。

著名的代数结构例子有实数、复数和四元数。与向量空间不同的是，任何给定维度大于一的非同构代数结构都是可能的。

物理学家感兴趣的一类代数结构是克利福德代数。克利福德代数是实数上的代数结构（作为向量空间是欧几里德空间），通过允许任意数量的形式量 e（类似于 i=√−1）与实数域相连。这些量的共同特点是每个量满足 e2=−1 或 e2=1。虽然低维度有很多关联代数结构，但只有其中几个是克利福德代数。实数形成唯一的一维克利福德代数，而由 e2=1 定义的双曲平面和由 e2=−1 定义的复平面是两个二维克利福德代数。双曲平面只是实数域的直积，意味着它的乘积是逐坐标的，(a,b)(c,d)=(ac,bd)，而复平面的乘法由(a,b)(c,d)=(ac−bd,ad+bc)定义。两个四维克利福德代数是 2×2 矩阵和四元数。2×2 矩阵包含零因子（乘积为零的非零矩阵），因此只形成一个环，而四元数不包含零因子，因此形成一个除环。然而，与复数不同，四元数不构成一个域，因为它们的乘法不满足交换律。复数的乘法使复平面成为一个交换除环，即一个域。

## 5. 代数化的数学

许多数学分支从代数的视角中受益。代数几何和代数组合学各自都有专门的期刊，而代数拓扑、代数逻辑和代数数论都有强大的追随者。许多其他更专业的数学领域也同样受益。

### 5.1 代数几何

代数几何始于我们在引言中提到的形状，例如直线 y=ax+b，圆 x2+y2=r2，球体 x2+y2+z2=r2，二次曲线 f(x,y)=0，其中 f 是关于 x 和 y 的二次多项式，二次曲面 f(x,y,z)=0，其中 f 再次是二次的，等等。

方便起见，我们可以将这些方程的两边移到左边，使得右边始终为零。然后，我们可以定义一个形状或_代数簇_，它由多项式的根或_零点_组成，或者更一般地说，由一组多项式的公共零点组成。

普通的分析几何或笛卡尔几何是在实数域上进行的。代数几何更常用于复数域，或者更一般地说，任何代数闭域上。用这种方式定义的簇被称为_仿射簇_。

有时候，代数闭包并不是可取的，例如在代数几何和数论的边界上工作时，可能会涉及到有限域或有理数。

许多种对象的特征在于它们的映射保持不变的结构。偏序集通过单调函数进行转换，保持顺序不变。代数通过同态变换，保持代数结构不变。在代数几何中，变量通过_正则_的 n 元_函数_ f:An→A 进行转换，这些函数在局部上是有理多项式。局部有理意味着在 f 的定义域的每个点上，存在一个邻域，在该邻域上 f 是两个多项式的比值，其中分母在该邻域中非零。

这个概念推广到了定义为 m 元的正则 n 元函数的正则函数 f:An→Am。

给定两个在 An 和 Am 中的变量 V 和 V′，从 An 到 Am 的正则函数，其在 V 上的限制是从 V 到 V′的函数，被称为_变量的正则函数_。仿射变量的范畴的对象是所有仿射变量，其态射是所有正则函数。

多项式是连续的，人们也会期望在代数变量之间存在连续的正规函数。然而，在代数变量的形状上会出现尖点、交叉和其他奇点的症状，这就产生了困难。在这里需要的是一种适当的拓扑结构来判断连续性。

关键在于不在仿射空间中工作，而是在其_射影空间_中工作。以欧几里得三维空间为例，其相关的射影空间是单位球，其中对径点被认为是同一点，形成一个二维流形。等价地，这是所有（无方向）通过原点的直线的空间。对于任意的仿射空间，其相关的射影空间是所有这样的直线的空间，被理解为一个流形。

适用于代数几何的射影空间拓扑是_Zariski 拓扑_，它的定义不是通过开集，而是通过闭集，闭集被认为是代数集，即由一组齐次多项式的公共零点构成的集合。关键定理是，仿射簇之间的正则映射在 Zariski 拓扑下是连续的。

### 5.2 代数数论

代数数论已经采纳了代数几何的这些推广。在数论中，一个特别重要的变体类别是椭圆曲线。

代数数论的一个著名成功是安德鲁·怀尔斯证明费马所谓的“最后定理”。这个问题在三个半世纪以来一直是一个悬而未决的问题。

### 5.3 代数拓扑学

代数拓扑学分析连通拓扑空间中的空洞和障碍。一个拓扑学家将所有物体想象成由坚固但非常柔软的橡皮泥制成，因此不需要区分咖啡杯和甜甜圈，因为它们可以相互转化。拓扑学关注具有 n 个手柄的咖啡杯、具有 n 个孔的表面以及更复杂的形状之间的相似性和差异。代数拓扑学通过同伦群和同调群来表达这些形状的不变量。

### 5.4 代数逻辑

代数逻辑在 1847 年的小册子中，布尔引入布尔代数，早早地开始了。现代代数的方法开始应用于布尔代数在 20 世纪。代数逻辑随后将其兴趣扩展到了一阶逻辑和模态逻辑。一阶逻辑中的中心代数概念是超乘积、元等价以及元和伪元变种。塔斯基的圆柱代数构成了一阶逻辑的特定抽象表述，其中对角关系编码相等和替换关系编码变量。模态逻辑作为一阶逻辑的一个片段，通过布尔模块使其成为代数。

## 6. 自由代数

给定任何系统，如整数算术或实数算术，我们可以用 T 表示由常量构建的所有确定术语的集合，构成确定语言，用 T\[V] 表示允许变量从集合 V 中取代一些常量符号的更大的不确定语言，其中包括诸如 x+(2/y) 这样的术语。当 V 只包含一个变量 “x” 时， T\[{“x”}] 通常缩写为 T\[“x”] 或只是 T\[x]，这通常是不含糊的。这个约定扩展到了术语代数 Φ 及其被视为组合术语的操作符列表；我们用 Φ\[V] 表示，并称之为 _术语代数_ 在 V 上。

术语代数的概念是一个纯粹的句法概念，只涉及某种语言的操作符号、常量和变量。术语 2+3 和 3+2 是不同的；同样，x+y 和 y+x 也是不同的术语。因此，它们可以被视为具体术语。

现在，在像整数这样的宇宙中，某些具体术语在变量的值不同的情况下总是求值为宇宙中的相同元素，例如 x+y 和 y+x。将等价的具体术语方便地收集到等价类中，每个等价类都被视为一个抽象术语。

作为抽象术语的简单例子，考虑形式为 ax+by 的线性多项式，其中 a 和 b 是非负整数，例如 7x+3y。所有这样的多项式的集合包括 0，并且在多项式加法下是封闭的，这是一种可结合和可交换的运算。因此，这个集合连同加法运算和零多项式构成了一个可交换的幂等半群。

这个幂等半群是自由代数的一个例子，即两个生成元 x 和 y 上的自由可交换幂等半群。它之所以是自由的，是因为它除了满足可交换幂等半群的规律外，没有其他规律。然而，它不是自由幺半群，因为它满足了可交换律。而两个生成元 x 和 y 上的自由幺半群实际上是由二元字母表{x,y}上的所有有限字符串组成的集合。

当交换律被引入为一条法则时，它将之前不同的字符串 xy 和 yx 标识为一个多项式；更一般地，任何具有相同数量的 x 和 y 的两个字符串都被标识为相同。

自由幺半群和自由交换幺半群是自由 C-代数的例子，其中 C 是代数类。在这两个例子中，类 C 分别是幺半群和交换幺半群。

一个自由的 C-代数是一种存在于语法和语义边界的代数。在语义方面，它是 C 的成员。在语法方面，它的元素的行为类似于受 C 的法则约束的项，但没有其他可以用其生成元表达的法则。可交换性 xy=yx 可以用两个生成元表达，因此两个或更多生成元的自由幺半群不能是可交换的，尽管一个生成元的自由幺半群，即由一个字母字母表上的所有有限字符串组成的集合，形成了一个可交换的幺半群。

在语法方面，集合 X 上的自由 C-代数 B 是由使用 C 的代数中的操作符号和常数形成的项代数的商。商集合将那些对于 C 的所有代数 A 和将 X 的变量赋值为 A 中的值时具有相同值的项标识为同一项。这样做足够满足 C 的每个法则（从而使得这个商集合成为一个 C-代数），同时仍然保留原始项代数的语法本质，这一点在下一段更加明确地说明。

（由于术语代数的概念在某些地方似乎有点循环，更详细的说明可能会澄清这个概念。给定语言 C，即常见于 C 代数的操作符号和常量符号，以及变量集合 X，我们首先形成代数的基础集合，然后将语言的符号解释为该集合中的操作和值。集合本身由使用操作符号从这些变量和常量符号通常方式构建的术语组成；从这个意义上说，这些元素是语法的。但现在我们改变了观点，将这些元素视为语义的，并且我们将语言的常量符号和操作符号视为需要在这个语义域（尽管是术语的语义域）中解释的语法实体，以将这组术语转化为术语代数。我们将每个常量符号解释为它本身。我们将每个 n 元操作符号 f 解释为将任意 n 个术语 t1,...,tn 作为其 n 个参数并返回单个术语 f(t1,...,tn)的 n 元操作。请注意，对 f 的这种解释只是返回一个术语，而不是实际构建它。所有的术语构建都是在我们生成代数的基础集合时完成的。）

从语义的角度来看，一个 C-代数 B 以及被视为变量的 B 的子集 X 被称为 X 上的自由 C-代数，或者是由 X 自由生成的，当给定任何 C-代数 A 时，X 中变量的任何在 A 中的赋值（即任何函数 f:X→A）唯一地扩展为一个同态 h:B→A。（当 h 限制在 X 上为 f 时，我们说 h:B→A 扩展了 f:X→A。）

作为一种方便的简写，没有生成元的自由 C-代数也可以称为初始 C-代数。初始 C-代数对每个 C-代数都有且仅有一个同态。

在继续示例之前，值得指出自由代数的一个重要基本性质，即从语义方面定义的性质。

_两个具有相同基数的生成集 X 和 Y 上的自由代数 B 和 B'是同构的。_

作为证明，选择任意的双射 f:X→Y。它的逆 f':Y→X 以及分别作用在 X 和 Y 上的两个恒等函数，构成了一个由四个函数组成的系统，这些函数在组合下是封闭的。每个函数都是从一个生成集到一个代数的映射，因此它们都有唯一的扩展到同态的映射。这四个同态映射也在组合下是封闭的。从 B 到自身的同态映射扩展了 X 上的恒等函数，因此必须是 B 上的恒等同态（因为后者存在且其在 X 上的限制是 X 上的恒等函数）。同样，从 G 到 G 的同态映射是一个恒等函数。因此，B 和 G 之间的同态映射可以以任意顺序组合为恒等映射，这使它们成为同构。而这正是 B 和 B'同构的意义所在。

这个事实使我们能够说给定集合上的_自由代数_，将同构的代数视为“道义上”相同。如果不是这样，我们的商构造将不完整，因为它产生了一个唯一的自由代数，而上述自由代数的定义允许将任何同构于商构造所产生的代数视为自由代数。由于 X 上的所有自由代数是同构的，商构造和任何其他构造一样好，并且还是证明它们存在的一种方式。它还确定了变量集的选择除了基数外是无关紧要的，正如直觉所暗示的那样。

### 6.1 自由幺半群和群

将 C 视为幺半群的类。由二元操作符和恒等常数符号确定的代数术语可以看作是带有变量和恒等常数符号副本的二叉树。根据结合性将树进行标识，将树展平为忽略操作应用顺序的单词（但不会颠倒任何参数的顺序）。这产生了由字母表 X 和恒等元素组成的单词。恒等律会消除恒等元素，除非单词仅由恒等符号组成，我们将其视为空单词。

因此，字母表 X 上的有限单词的幺半群是 X 上的自由幺半群。

另一种表示自由幺半群的方法是作为一棵无限树，其中每个顶点都有 n 个后代，每个后代对应字母表中的一个字母，并且每条边都标有相应的字母。每个顶点 v 表示由从根到 v 的路径上遇到的字母组成的单词。将 u 和 v 连接起来的结果是通过取以顶点 u 为根的子树到达的顶点，注意到这棵树同构于完全树，并将 v 定位在这棵子树中，就好像它是完全树一样。

如果我们忽略这棵树中边的方向和标签，我们仍然可以确定根：它是唯一一个有 n 条边与之关联的顶点，所有其他顶点都有 n+1 条边，即一条入边和 n 条出边。

在一个集合上的自由交换幺半群是那个幺半群，其生成元的行为就像自由幺半群一样（特别是它们仍然是原子），但满足额外的法则 uv=vu。我们进一步进行识别，例如“dog”和“dgo”。单词中字母的顺序现在不重要了，重要的是每个字母有多少个副本。这个信息可以表示为自然数的 n 元组，其中 n 是字母表的大小。因此，n 个生成元的自由交换幺半群是 Nn，即自然数的 n 元组在加法下的代数。

它也可以从自由幺半群的树表示中通过识别顶点来获得。考虑两个字母的情况 n=2。由于识别不改变单词长度，所有的识别都是在与根节点相同深度的顶点上进行的。我们同时进行所有的识别，方法如下。在每个顶点 v 处，识别 v01 和 v10 及其子树。在以前，深度为 n 的顶点有 2n 个，现在有 n+1 个。此外，我们不再有一棵树，而是有平面的右上象限，即 N2，顺时针旋转 135 度，每个顶点 v 位于一个菱形的顶部，其其他顶点 v0 和 v1 位于下一层，并且识别的一对 v01=v10 位于两者之下。

为了形成 n 个生成元的自由群，首先形成 2n 个生成元的自由幺半群，其中生成元被组织成互补对，每个对都是对方的逆元，然后从所有单词中删除所有相邻的互补对。

这个观点并不特别有洞察力。树表示的群对应物更好地呈现了自由群。考虑 n=2 个生成元 A 和 B 的自由群。我们从 4 个生成元 A、B、a、b 的自由幺半群开始，其中 a 是 A 的逆元，b 是 B 的逆元。这棵树的每个顶点都有 4 个后代。所以根节点的度为 4，其余顶点的度为 5：除了根节点外，每个顶点都有一条指向内部的边，比如生成元 a，和四条指向外部的边。考虑任意非根节点 v。删除相邻的互补对的效果是将 v 的直接祖先与 v 的四个后代之一进行等同，即使得从祖先到后代的路径成为互补对。对于每个非根节点 v，这些等同将 v 的度从 5 降低到 4。根节点的度仍然为 4。

所以现在我们有一个无限图，每个顶点的度数都是 4。与自由幺半群的树不同，其中根与其他顶点在拓扑上是不同的，自由群的树是完全均匀的。因此，如果我们丢弃顶点标签，仅依靠边标签进行导航，任何顶点都可以被视为群的单位元。

这种均匀性对于具有 2 个生成元的自由阿贝尔群仍然成立，其顶点仍然具有度数 4。然而，额外的等同关系将其从树（没有循环的图）转变为一个网格，其顶点是平面上的格点。也就是说，具有 2 个生成元的自由阿贝尔群是 Z2，具有 n 个生成元的自由阿贝尔群是 Zn。边是连接相邻格点的线段。

### 6.2 自由环

没有生成元的自由幺半群、自由群和自由环都是只包含加法单位元 0 的单元代数。一个带单位元的环意味着具有乘法单位元，即一个字母 ε。但是这使得 ε 成为环的加法群的一个生成元，而一个生成元的自由阿贝尔群是整数。因此，没有生成元的带单位元的自由环就是整数，包括加法和乘法。

自由环在一个生成元 x 上必须包括 x2、x3 等通过乘法得到的项，但这些项可以相加和相减，从而得到多项式，例如 7x3−3x2+2x，但没有常数项，除了 0 本身。环的分配律意味着像(7x+x2)(2x3+x)这样的项可以展开为 7x2+x3+14x4+2x5。现在应该清楚了，这些只是没有常数项的普通多项式；特别是我们缺少零次多项式 1，因此这个环没有乘法单位元。然而，即使我们没有指定，它仍然是一个可交换的环。带有一个生成元的带单位元的自由环引入了 1 作为乘法单位元，并成为普通的一元多项式，因为现在我们可以形成所有的整数。就像幂半群一样，带有两个生成元的带单位元的自由环不是可交换的，多项式 xy 和 yx 是不同的。然而，带有两个生成元的带单位元的自由_可交换_环由整数上的普通二元多项式组成。

### 6.3 自由组合结构

从迄今为止的例子中，人们可能得出结论，一个或多个生成器的所有自由代数都是无限的。这绝不总是如此；作为反例，我们可以指出一些类别：集合、指向集合、双指向集合、图、无向图、布尔代数、分配格等。每个类别都形成了之前定义的局部有限的变种。

指向集合是一个带有一个常数的代数，比如 c。在 x 和 y 上的自由指向集合有三个元素，x、y 和 c。双指向集合是一个带有两个常数 c 和 d 的代数，然后在 x 和 y 上的自由双指向集合有四个元素，x、y、c 和 d。

在自动机理论中出现的有向图，其中多条边可以连接相同的两个顶点，可以组织为具有两个一元运算 s 和 t 的代数，满足 s(s(x))=t(s(x))=s(x)和 t(t(x))=s(t(x))=t(x)。一个生成器 x 的自由图有三个元素，x，s(x)和 t(x)，分别构成一条边及其两个端点或_顶点_。在这个框架中，顶点是满足 s(x)=x 的元素（因此 t(x)=x，因为 x=s(x)=t(s(x))=t(x)）；所有其他元素构成边。n 个生成器的自由图由 n 条这样的边组成，全部独立。其他图通过识别元素而产生。识别一条边与另一条边或一个顶点没有意义，因为这只是将第一条边吸收到第二个实体中。这样只剩下顶点；识别两个顶点会产生一个同时属于两条边或在识别 s(x)=t(x)时产生一个自环的单个顶点。

术语“有向”比“定向”更可取，因为在组合数学中理解的有向图是具有额外属性的有向图，即如果 s(x)=s(y)且 t(x)=t(y)，则 x=y；也就是说，在给定方向上两个顶点之间只允许一条边。

未定向图定义为具有额外一元操作 g 的图，满足 g(g(x))=x 和 s(g(x))=t(x)（因此 s(x)=s(g(g(x)))=t(g(x))）。x 上的自由无向图由 x、s(x)、t(x)和 g(x)组成，其中对 x,g(x)构成了一个双向车道的两个单向车道，位于 s(x)=t(g(x))和 t(x)=s(g(x))之间。无向图元素的识别与其定向对应物相同：只有识别顶点才有意义。然而，这里有一个有趣的变化：顶点可以分为两种，满足 x=g(x)的顶点和不满足的顶点。后一种顶点现在是非对称的：双向边的一个方向与其顶点相对应，而另一个方向形成一个定向环，其另一个方向是一个顶点。对于满足“如果 s(x)=s(y)且 t(x)=t(y)则 x=y”的无向图，不会出现这种现象。

### 6.4 自由逻辑结构

布尔代数传统上被公理地定义为互补分配格，这有助于显示它们形成一个种类，而且是一个有限公理化的种类。然而，布尔代数在其本身中是如此基础，以至于与其为此目的定义格、分配和互补，不如从初始布尔代数中获得它们更容易且更有洞察力。只需将其定义为二元集合{0,1}，常数（零元操作）0 和 1，以及 16 个二元操作。然后，任何具有这 16 个操作和两个满足初始布尔代数方程的常数的代数都是布尔代数。

布尔代数类的一个几乎确定的特性是，它们在初始布尔代数中的多项式是该代数上的所有操作。问题在于，只包含一个元素或不一致代数的不一致类也具有这个特性。然而，通过添加布尔代数是一致的这一条件，可以轻松地排除这个类。但是，只是勉强地——在布尔代数中添加任何新方程（不引入新操作）就可以公理化不一致代数。

Sheffer 已经证明，常数和 16 个操作可以作为多项式生成，只需要一个常数（可以是 0 或 1）和一个二元操作（可以是 NAND，¬(x∧y)，或 NOR，¬(x∨y)）。任何这样的充分集合被称为“基础”。沿着同样的思路，Stone 已经证明，合取、异或和常数 1 构成了一个基础。Stone 基础的重要性在于，用这些操作组织的布尔代数满足了作为乘法的合取和作为加法的异或的交换环的所有公理，以及定律 x2=1。满足这个最后条件的环被称为“布尔环”。布尔环与布尔代数等价，因为它们具有相同的多项式。

布尔代数的原子是一个元素 x，对于所有的 y，x∧y 要么是 x，要么是 0。无原子的布尔代数是没有原子的布尔代数。

对于每个有限的 2 的幂次，存在一个基数为该幂次的布尔代数，它与该基数的幂集 2X 上的布尔代数同构，其中 X 是集合，操作为并集、交集和相对于 X 的补集。因此，所有有限的布尔代数的基数都是 2 的幂次。这种情况在无限的布尔代数中发生了变化；特别是存在可数的布尔代数。其中一个例子是可数个生成元上的自由布尔代数，它是唯一的可数无原子布尔代数。自然数集 N 上的有限和余有限（有限集的补集）子集构成了幂集布尔代数 2N 的一个子代数，它与自由布尔代数不同构，但它有原子，即单元素集合。

在具有 n 个生成元的自由布尔代数 F(n)中，包含了所有对于两元素布尔代数的 22n 个 n 元操作。因此，布尔代数形成了一个局部有限的变种。

分配格的等式理论是通过从布尔代数的等式理论中选择仅具有单调二元运算的操作（省略常数）而得到的。这些操作具有以下特性：如果其中一个参数从 0 变为 1，结果不会从 1 变为 0。分配格是那些仅由单调二元运算构建的项之间的布尔等式的任何模型。因此，每个布尔代数都是一个分配格。

分配格可以是任意“薄”。在极端情况下，任何链（线性或全序，例如实数的标准排序）在最大和最小的常规操作下形成一个分配格。由于我们省略了常数，这包括了空格，我们在这里没有将其排除作为一个代数。 （一些作者不允许空集作为代数，但这种禁止会破坏许多好的定理而没有获得任何有用的定理。）因此，存在各种可能基数的分配格。

每个有限维向量空间都是自由的，可以由任意选择的基生成。只要我们接受选择公理，这个结论也适用于无限维向量空间。因此，当标量乘法被组织为每个域元素的一元操作时，有限域上的向量空间形成了一个局部有限的变种。

### 6.5 范畴论中的自由代数

我们现在从范畴论的角度来考虑自由代数的组织方式。我们定义由集合 X 生成的自由代数 B 具有以下性质：对于每个代数 A 和每个估值 f:X→A，存在唯一的同态 h:B→A。现在每个同态 h:B→A 都必然以这种方式产生，因为它在 X 上的限制作为从 X 到 A 的函数是一个估值。此外，每个函数 f:X→A 都可以作为其扩展到同态的限制在 X 上的形式出现。因此，我们在从 X 到 A 的函数和从 B 到 A 的同态之间有一个双射关系。

现在这里的类型有点随意，让我们整理一下。由于 X 是一个集合而 A 是一个代数，f 最好被类型化为 f:X→U(A)，其中 U(A)表示 A 的底层集合。而 X 与 B 的关系最好用符号 B=F(X)来理解，表示由集合 X 生成的自由代数。因此，U 将代数映射到集合，而 F 将集合映射到代数。F 和 U 一般来说不是彼此的逆映射，但它们在某种程度上是相关的，我们现在将其明确化。

对于任何一个范畴 C，通常使用符号 C(A,B) 来表示范畴 C 中从对象 A 到对象 B 的所有态射的集合。而从集合 X 到集合 Y 的所有函数的集合可以被理解为这个约定的特例 Set(X,Y)，其中 C 被取为所有集合的类 Set，我们可以将其视为“离散”代数，也就是没有结构的代数。一类代数以及其成员之间的指定的同态构成了一个“范畴”的实例。这个类的成员被称为“范畴的对象”，而同态被称为“态射”。

我们刚刚观察到的双射现在可以表述为

C(F(X),A)≅Set(X,U(A))

这样的双射被称为 Set 和 C 之间的一个_伴随_。这里的 F:Set→C 和 U:C→Set 分别是这个伴随的左和右_伴随_；我们说 F 是 U 的左伴随（或者说 F 是 U 的伴随）而 U 是 F 的右伴随。

我们只是描述了 F 如何将集合映射到代数，以及 U 如何将代数映射到集合。然而，F 还将函数映射到同态，将每个函数 f 映射到其唯一的同态扩展，而 U 将同态映射到函数，即将同态本身作为函数。这样的映射称为_函子_。

一般来说，一个范畴 C 由对象 a、b、c 和态射 f:a→b 组成，还有一个关于“可组合”态射 f:b→c、g:a→b 的关联组合法则，得到态射 fg:a→c。此外，每个对象 a 都有一个恒等元素 1a:a→a，当与一个态射 f（在一侧或另一侧）可组合时，与之组合得到 f。一个函子 F:C→D 将 C 的对象映射到 D 的对象，将 C 的态射映射到 D 的态射，使得 F(fg)=F(f)F(g)和 F(1a)=1F(a)。也就是说，函子是“范畴的同态”，保持组合和恒等性。

没有进一步的限定，这样的类别被认为是一个_抽象_类别。我们一直在使用的类别是_具体_的，因为它们带有给定的底层集合或_遗忘_函子 U:C→Set。也就是说，代数是基于集合的，同态是这些集合之间的某些函数，而 U 只是“忘记”了代数结构。这样的遗忘函子在以下意义上是_忠实_的：对于 C 中的任意两个态射 f,g:a→b，如果 U(f)=U(g)，那么 f=g，即 U 不会将不同的同态等同起来。一般来说，具体类别被定义为一个类别 C 和一个忠实的遗忘函子 U:C→Set。

类别本身可以进一步推广为二范畴，作为二维图形上的代数，其中一维细胞的结合被推广为二维细胞的二维结合。进一步简化自由代数机制，即通过抽象伴随关系作为类别中同构的自然二维对应，而类别又是集合中元素相等的自然一维对应，即两个点可以变成一个点的零维概念。这导致了抽象单子的概念，它只是一个伴随一维细胞对的组合，其中一个是抽象化函子 F，它从 V 制造出自由代数 F(V)的一维细胞。普通或具体单子是作为类别的具体一维细胞的函子组合而产生的，这些类别是类别的二范畴中的具体一维细胞。
