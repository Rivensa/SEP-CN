# 类比与类推推理 analogy and analogical reasoning (Paul Bartha)


*首次发表于2013年6月25日；实质性修订于2019年1月25日*

*类比*是指两个对象或对象系统之间的比较，强调它们被认为相似的方面。*类比推理*是依赖于类比的任何类型的思维。*类比论证*是对类比推理形式的明确表述，引用了两个系统之间的已接受的相似之处，以支持某种进一步的相似性存在的结论。一般来说（但并不总是如此），这样的论证属于扩大推理的范畴，因为它们的结论并不确定，只是以不同程度的强度支持。然而，关于类比论证的正确描述存在争议（参见[§2.2](https://plato.stanford.edu/entries/reasoning-analogy/#Cha)）。

类比推理对人类思维至关重要，可以说对某些非人类动物也是如此。历史上，类比推理在广泛的问题解决环境中发挥了重要但有时神秘的作用。自古以来，明确使用类比论证一直是科学、哲学和法律推理的一个独特特征。本文主要关注类比论证的性质、评估和证明。相关主题包括[隐喻](https://plato.stanford.edu/entries/metaphor/)、[科学模型](https://plato.stanford.edu/entries/models-science/)以及[法律推理中的先例和类比](https://plato.stanford.edu/entries/legal-reas-prec/)。

## 1. 赫尔米亚的阿莫尼俄斯

类比被广泛认为在发现过程中起着重要的启发作用。它们在各种环境中被广泛应用，并取得了相当大的成功，用于产生洞察力并制定可能的问题解决方案。根据化学和电学先驱约瑟夫·普里斯特利的说法，

> 类比是我们在所有哲学研究中的最佳指南；所有不是纯粹偶然发现的发现都是在它的帮助下实现的。（1769/1966: 14）

Priestley可能夸大了这个案例，但毫无疑问，类比已经在许多领域中提出了有益的研究方向。由于其启发式价值，类比和类比推理一直是人工智能研究的特别关注点。Hájek（2018）研究了类比作为哲学中的一种启发式工具。

类比具有相关（并且不完全可分离的）*证明*角色。这个角色在明确提供类比论证以支持某个结论的情况下最为明显。对于结论的支持程度可以有很大的变化。在一个极端情况下，这些论证可以具有很强的预测性。例如：

***例子1***。水动力学类比利用理想流体流动和扭转问题的方程之间的数学相似性。为了预测计划结构中的应力，可以构建一个流体模型，即一个水通过的管道系统（Timoshenko和Goodier 1970）。在理想化的限制下，这种类比使我们能够进行演示性推理，例如，从流体模型中的测量量到扭转问题中的类似值。在实践中，存在许多复杂因素（Sterrett 2006）。

在另一个极端，类比论证可能对其结论提供非常弱的支持，仅仅建立了最低限度的可信度。考虑：

***例子2***。托马斯·里德（1785年）关于其他星球上存在生命的论证（斯特宾1933年；米尔1843/1930年；罗宾逊1930年；科皮1961年）。里德指出地球和我们太阳系中的其他行星之间存在许多相似之处：它们都绕太阳运行并受其照射；几个行星有卫星；它们都绕轴旋转。因此，他得出结论：“认为那些行星可能像我们的地球一样，是各种生物的栖息地，这并不是不合理的”（1785年：24）。

这种谦逊并不罕见。通常类比论证的目的只是说服人们认真对待一个想法。例如：

***例子3***. 达尔文认为他在使用人工选择和自然选择之间的类比来论证后者的可信性：

> 为什么我不能提出自然选择的假设（根据家养动物的类比以及我们对生存斗争和有机生物可变性的了解，这个假设在某种程度上是有可能的），并尝试看看这个自然选择的假设是否能够解释（我认为它能够解释）大量的事实……（《给亨斯洛的信》，1860年5月，达尔文1903年）

这里显然，达尔文自己承认，他的类比是用来显示这个假设在某种程度上是可能的，因此值得进一步调查。然而，有些人拒绝对达尔文的推理进行这种描述（Richards 1997; Gildenhuys 2004）。

有时类比推理是一种唯一可用的假设证明形式。*民族志类比*方法用于解释

根据他们的工艺品与现代人使用的相似性，推测考古遗址（或古代文化）的古代居民的不可观察行为。（Hunter和Whitten 1976：147）

***例子4***。Shelley（1999, 2003）描述了如何使用民族志类比来确定在秘鲁安第斯山脉发现的莫切陶罐颈部的奇怪标记的可能意义。秘鲁的当代陶工使用这些标记（称为“sígnales”）来表示所有权；这些标记使他们能够在几个陶工共用一个窑炉或储存设施时重新获得自己的作品。在这种情况下，类比推理可能是对过去进行推断的唯一途径，尽管这一点存在争议（Gould和Watson 1982；Wylie 1982, 1985）。类比推理对于由于观察限制而无法接近的宇宙现象可能具有类似的重要性（Dardashti等，2017）。有关进一步讨论，请参见[§5.1](https://plato.stanford.edu/entries/reasoning-analogy/#AnaCon)。

正如库恩（1996）等哲学家和历史学家反复指出的那样，我们所确定的两个角色，即发现和证明，并不总是清晰分开的。实际上，在类比的“规划”（或“范例”）角色中，这两个功能是融合在一起的：在一段时间内，类比可以塑造研究计划的发展。例如：

***例子5***。十九世纪某些物理学家在研究光谱线时，多年来采用了一种“声学类比”。人们认为离散光谱与声学情况完全类似，原子（和/或分子）作为振荡器，以共振音叉的方式产生或吸收振动。（Maier 1981: 51）

在这个类比的指导下，物理学家寻找了一组展现出谐振子特征频率模式的光谱线。这个类比不仅用于支持猜想的合理性，还通过指引科学家朝着特定方向进行研究来引导和限制发现。

更一般地说，类比可以通过指导概念发展起到重要的规划作用（参见[§5.2](https://plato.stanford.edu/entries/reasoning-analogy/#ConChaTheDev)）。在某些情况下，一个规划性的类比最终会导致两个不同研究领域的理论统一。

***例子6*.** 笛卡尔（1637/1954）之间几何和代数之间的关系提供了一种系统处理长期被认为是类似的几何问题的方法。当程序性类比崩溃时，类比与发现之间存在着非常不同的关系，音响类比的最终命运就是这样。 随着量子理论的出现，人们清楚地认识到原子光谱具有完全不同的解释。在这种情况下，新颖的发现出现在由指导类比塑造的背景期望之**中**。 还有第三种可能性：一个无效或误导性的程序性类比可能会简单地根深蒂固并自我持续，因为它导致我们“构造…符合它的数据” (Stepan 1996: 133)。可以说，这第三种可能性的危险为发展对类比推理和类比论点给出了强有力的动机。

*类比认知*，它涵盖了发现、构建和使用类比所涉及的所有认知过程，比类比推理（Hofstadter 2001; Hofstadter and Sander 2013）更广泛。理解这些过程是当前认知科学研究的一个重要目标，也是一个产生许多问题的目标。人类如何识别类比？非人类动物是否以类似于人类的方式使用类比？类比和隐喻如何影响概念形成？

然而，本文专注于类比论证。具体而言，它关注三个中心认识论问题：

1. 我们应该使用什么标准来评估类比论证？2. 对于类比推理，可以提供什么哲学上的理据？3. 类比论证如何融入更广泛的推理背景（即，我们如何将其与其他形式的推理，特别是理论确认，结合起来）？

在对类比论证的基本结构进行初步讨论之后，本文回顾了一些试图回答这三个问题的尝试。找到这些答案将构成理解类比推理性质的重要第一步。然而，将这些问题孤立起来，就是做出一个非平凡的假设，即存在一个“类比论证的理论”——正如我们将看到的，这个假设受到哲学家和认知科学家的不同方式的质疑。

## 2. 类比论证

 ### 2.1 例子

赫尔米亚的阿莫尼俄斯的主题、强度和逻辑结构各不相同。为了欣赏这种多样性，增加我们的例子库是有帮助的。首先，一个几何学的例子：

***例子7***（矩形和盒子）。假设你已经确定，在固定周长的所有矩形中，正方形具有最大的面积。类比地，你猜测，在固定表面积的所有盒子中，立方体具有最大的体积。

科学史上的两个例子：

***例子8***（吗啡和哌替啶）。1934年，药理学家Schaumann正在测试合成化合物的抗痉挛效果。这些药物的化学结构与吗啡相似。他观察到其中一种化合物——哌替啶，也被称为Demerol——对小鼠产生了一种之前只有吗啡才能引起的物理效应：它引起了一种S形尾巴弯曲。通过类比，他推测这种药物可能也具有吗啡的麻醉效果。对大鼠、兔子、狗和最终人类的测试表明，哌替啶像吗啡一样是一种有效的止痛药（Lembeck 1989: 11; Reynolds and Randall 1975: 273）。

***例子9***（普里斯特利关于静电力的观察）。1769年，普里斯特利提出，在一个带电的空心球壳内部没有电场影响的现象是电荷以倒数平方力相互吸引和排斥的证据。他通过引用均匀密度的空心球壳内部没有重力的类似情况来支持他的假设。

最后，一个来自法律推理的例子：

***例子10***（合理关怀义务）。在一起被广泛引用的案例中（*Donoghue v. Stevenson* 1932 AC 562），英国上议院裁定一家姜汁啤酒制造商对一名消费者因瓶中有只死蜗牛而导致生病承担赔偿责任。法院认为制造商有责任在制造可能会对消费者造成伤害的产品时“合理关怀”，在没有这种关怀的情况下，消费者没有中间检查的可能性。这个著名案例中阐述的原则被类比地扩展到一家工程公司，该公司的疏忽修理工作导致电梯坍塌，允许对其造成的伤害进行赔偿（*Haseldine v. CA Daw & Son Ltd.* 1941 2 KB 343）。相比之下，这个原则不适用于一个工人因为一台有缺陷的起重机而受伤的案例，因为工人有机会检查起重机，甚至知道其中的缺陷（*Farr v. Butters Brothers & Co.* 1932 2 KB 606）。

 ### 2.2 特征化

所有这些例子有什么共同之处（如果有的话）？我们从一个简单的准正式特征化开始。类似的表述可以在初级批判性思维文本中找到（例如，Copi和Cohen 2005），也可以在论证理论的文献中找到（例如，Govier 1999，Guarini 2004，Walton和Hyra 2018）。类比论证具有以下形式：

1. 在某些（已知的）方面，S与T相似。 2. S具有进一步的特征Q。 3. 因此，T也具有特征Q，或者具有与Q相似的特征Q∗。

（1）和（2）是前提。 （3）是论证的结论。 这个论证形式是*扩大性的*； 结论不一定能从前提中得出。

S和T分别被称为“源领域”和“目标领域”。一个“领域”是一组对象、属性、关系和函数，以及关于这些对象、属性、关系和函数的一组可接受的陈述。更正式地说，一个领域由一组对象和关于它们的一组解释陈述组成。这些陈述不一定属于一阶语言，但为了简单起见，这里使用的任何形式化都将是一阶的。我们使用无星号的符号（a，P，R，f）来指代源领域中的项目，使用带星号的符号（a∗，P∗，R∗，f∗）来指代目标领域中的相应项目。在[*Example 9*](https://plato.stanford.edu/entries/reasoning-analogy/#Example9)中，源领域的项目涉及到引力；目标领域的项目涉及到静电吸引。

从形式上讲，S和T之间的“类比”是S中的对象、属性、关系和函数与T中的对象、属性、关系和函数之间的一对一映射。并不是S和T中的所有项目都需要对应。通常，类比只确定一组选定项目之间的对应关系。在实践中，我们通过指出最显著的相似之处（有时也包括差异）来指定一个类比。

我们可以通过引入赫尔米亚的阿莫尼俄斯（Hesse，1966）中的*表格表示*来改进对类比论证的初步描述。我们将相应的对象、属性、关系和命题并列放在一个包含两列的表格中，每个领域占据一列。例如，里德的论证（[*例子2*](https://plato.stanford.edu/entries/reasoning-analogy/#Example2)）可以表示如下（使用⇒表示类比推理）：

| | 地球（S） | | 火星（T） | | --- | --- | --- | --- | | **←垂直→** | 已知相似性： | | | | 绕太阳运行 | ←水平→ | 绕太阳运行 | | | 有卫星 | 有卫星 | | | | 自转 | 自转 | | | | 受重力影响 | 受重力影响 | | | | 推断出的相似性： | | | | | | 支持生命 | ⇒ | *可能*支持生命 |

 图1.

赫尔米亚的阿莫尼俄斯基于这种表格表示引入了有用的术语。类比中的*水平关系*是在领域之间的映射中的相似性（和差异）关系，而*垂直关系*是在每个领域内的对象、关系和属性之间的关系。地球有月亮和火星有卫星之间的对应关系（相似性）是一种水平关系；拥有月亮和支持生命之间的因果关系是源领域内的一种垂直关系（目标领域中可能存在一个不同的关系）。

在早期对类比的讨论中，Keynes（1921）引入了一些术语，这些术语也很有帮助。

***正类比***。假设P代表关于源领域S的一系列被接受的命题P1，...，Pn。假设相应的命题P∗1，...，P∗n，简称为P∗，都被接受为适用于目标领域T，以便P和P∗表示被接受的（或已知的）相似之处。那么我们将P称为*正类比*。

***负向类比***。假设A代表在S中被接受为成立的命题列表A1,...,Ar，B∗代表在T中成立的命题列表B∗1,...,B∗s。假设类比命题A∗=A∗1,...,A∗r在T中不成立，类似地，命题B=B1,...,Bs在S中不成立，因此A,∼A∗和∼B,B∗表示被接受（或已知）的差异。我们将A和B称为*负向类比*。

***中性类比***。*中性*类比由关于S的被接受命题组成，不知道在T中是否存在类似的命题。

 最后我们有：

***假设类比***。假设类比就是我们关注的中性类比中的命题Q。

这些概念使我们能够为个体类比论证提供一个比原始论证更丰富的描述。

（4）增强的表格表示 源（S）目标（T） PP∗[正向类比]A∼A∗[负向类比]∼B B∗QQ∗（可能）

一个类比论证可以这样总结：

在目标领域中，Q∗的成立是合理的，*因为*它与源领域存在某些已知（或被接受的）相似之处，*尽管*存在某些已知（或被接受的）差异。

为了使这种描述有意义，我们需要对“可信度”这个词的含义做一些解释。为了确保广泛适用于强度差异很大的类比论证，我们将“可信度”解释得相对宽松，意思是“有一定程度的支持”。一般来说，在提出主张之后，但在进行严格测试或证明之前，人们会对可信度进行判断。下一小节将进一步讨论。

请注意，这种描述在许多方面是不完整的。我们列举相似之处和不同之处的方式，领域之间的对应关系的性质：这些都没有具体说明。这种描述也无法适应使用多个类比（即多个源领域）进行推理的情况，而这在法律推理中普遍存在，在其他领域也很常见。然而，要更充分地描述这种论证形式，要么需要朝着类比推理的实质性理论迈进一步，要么需要将注意力限制在某些类比论证的类别上。

类比论证在论证理论中被广泛讨论。关于它们是否构成演绎推理的一种形式存在着相当大的争议（Govier 1999; Waller 2001; Guarini 2004; Kraus 2015）。论证理论家还利用诸如言语行为理论（Bermejo-Luque 2012）、论证模式和对话类型（Macagno et al. 2017; Walton and Hyra 2018）等工具来区分不同类型的类比论证。

类比论证也在科学模型和基于模型的推理的广泛文献中得到讨论，这一领域的研究源于赫斯（Hesse）（1966）的先驱。贝勒-琼斯（Bailer-Jones）（2002）对类比和模型进行了有益的区分。虽然“许多模型源于类比”（2002: 113），并且类比“可以作为催化剂帮助建模”，但贝勒-琼斯观察到“建模的目的与类比本质上无关”。简而言之，模型是预测和解释的工具，而类比论证旨在建立合理性。类比根据源-目标的相似性进行评估，而模型根据其“提供对现象的访问方式，即解释有关现象的可用经验数据”的成功程度进行评估。然而，如果我们将视野扩展到除了类比论证之外的领域，模型与类比之间的联系就会恢复。例如，纳西西安（Nersessian）（2009）强调了类比模型在概念形成和其他认知过程中的作用。

 ### 2.3 可信度

说一个假设是可信的，意味着它有认识论的支持：我们有一些理由相信它，即使在测试之前。在调查的背景下，可信度的断言通常也具有实用的含义：说一个假设是可信的意味着我们有一些理由进一步调查它。例如，一个数学家在证明中工作时，如果一个猜想“有一些成功的机会”，他会认为它是可信的（Polya 1954（v. 2）：148）。在这两个观点上，关于可信度的断言是分类的还是程度的存在歧义。这些观察结果指出了可信度的两个不同概念的存在，即*概率*和*模态*，其中任何一个都可能反映类比论证的预期结论。

根据概率的概念，合理性自然地与理性信任（理性主观信念程度）等同，并且通常表示为概率。在《逻辑学体系》中，米尔对类比论证进行了经典的阐述：

> 毫无疑问，每一种[未知是否无关]相似性都会为结论提供一定程度的概率，超出了原本存在的概率。(Mill 1843/1930: 333)

在[§2.2](https://plato.stanford.edu/entries/reasoning-analogy/#Pla)中引入的术语中，米尔的观点是正面类比的每个元素都增加了结论的概率。当代的“结构映射”理论([§3.4](https://plato.stanford.edu/entries/reasoning-analogy/#ForCriStrMapThe))采用了一种受限制的版本：两个领域之间的每个*结构*相似性都对整体相似度的度量以及类比论证的强度有贡献。

在另一种*模态*概念中，“p是合理的”不是一个程度问题。大致上，意思是有足够的初步理由来认真对待p，即进一步调查（可行性和兴趣的限制下）。非正式地说，p通过了初步筛选程序。没有程度的断言。相反，“合理的”可以被看作是一个认识模态运算符，旨在捕捉一个概念，即*表面上*的合理性，它比普通的认识可能性稍强。其目的是从一堆保持裸露的认识可能性的想法中单独出来p。举例来说：在1769年，普里斯特利的论证([*Example 9*](https://plato.stanford.edu/entries/reasoning-analogy/#Example9))，如果成功，将为静电吸引力的反比平方定律建立*表面上*的合理性。*认识*可能性的集合——与当时的知识相容的关于静电吸引力的假设——要大得多。数学中的个别类比论证（例如[*Example 7*](https://plato.stanford.edu/entries/reasoning-analogy/#Example7)）几乎总是针对*表面上*的合理性。

模态概念在一些关于类比推理的讨论中起着重要作用。物理学家N.R.坎贝尔（1957）写道：

> 但是为了使一个理论有价值，它必须展示出一种类比。假设的命题必须与某些已知的法则相类似....（1957: 129）

谈到傅立叶的热传导理论中类比的作用，坎贝尔写道：

> *某种*类比对于它是必不可少的；因为只有这种类比才能将这个理论与其他可能用来解释相同规律的理论区分开来。（1957年：142）

这里有一个有趣的概念，那就是“有价值的”理论。我们可能不同意坎贝尔的观点，即类比的存在对于一个新理论的“有价值性”是“必要的”。但考虑到一个较弱的论点，即一个可接受的类比足以证明一个理论是“有价值的”，或者（更进一步地）一个可接受的类比为认真对待该理论提供了可推翻的理由。（可能的推翻因素可能包括内部不一致性、与已接受的理论不一致性，或者存在一个（明显更优越的）竞争性类比论证。）关键在于，坎贝尔追随19世纪的哲学家科学家（如赫歇尔和惠威尔）的引导，认为类比可以建立这种“表面上的”合理性。斯奈德（2006）对后两位思想家及其关于类比在科学中的作用的观点进行了详细讨论。

一般来说，类比论证可以用来建立其结论的任一种合理性；它们可以具有概率使用或模态使用。[*例子7*](https://plato.stanford.edu/entries/reasoning-analogy/#Example7)到[9](https://plato.stanford.edu/entries/reasoning-analogy/#Example9)最好解释为支持模态结论。在这些论证中，类比被用来表明一个猜想值得认真对待。坚持将结论放在概率的术语中会分散对论证要点的注意力。结论可以被建模（由贝叶斯学派）为具有一定的概率值，因为它被认为是“表面上的”合理的，但反之则不然。[*例子2*](https://plato.stanford.edu/entries/reasoning-analogy/#Example2)可能被认为主要是针对概率性结论的。

两个概念之间应该有联系。事实上，我们可能认为同样的类比论证可以为假设建立* prima facie *的合理性和一定程度的概率。但是，在认识模态概念和概率之间进行翻译是困难的（Cohen 1980; Douven and Williamson 2006; Huber 2009; Spohn 2009, 2012）。我们不能简单地将概率概念作为原始概念。将这两种合理性概念分开似乎是明智的。

### 2.4 类比推理规则？

[Schema (4)](https://plato.stanford.edu/entries/reasoning-analogy/#schema4) 是一个代表所有类比论证（无论好坏）的模板。它不是一个推理规则。尽管特定的类比论证被提出时充满自信，但至今没有人提出过可接受的规则或一套规则来进行有效的类比推理。甚至没有一个合理的候选者。这种情况与演绎推理以及诸如枚举归纳等基本形式的归纳推理形成鲜明对比。

当然，很难证明永远不会有成功的类比推理规则被提出。但考虑以下候选者，使用了模式（4）的概念，并且只是在基本特征的基础上迈出了一小步。

(5)

假设S和T是源领域和目标领域。假设P1，...，Pn（其中n≥1）表示正向类比，A1，...，Ar和∼B1，...，∼Bs表示（可能是空的）负向类比，Q表示假设类比。在没有其他理由的情况下，推断在目标领域中以支持度p>0的程度成立Q∗，其中p是n的增函数和r、s的减函数。

规则（5）是基于[列举归纳的直接规则](https://plato.stanford.edu/entries/induction-problem/)的模型，并受到米尔对类比推理的观点的启发，如[§2.3](https://plato.stanford.edu/entries/reasoning-analogy/#Pla)所述。我们使用“支持度”的通用短语来代替概率，因为除了类比论证之外，还有其他因素可能影响我们对Q∗的概率分配。

很明显，(5)是一个不可行的方案。主要问题在于该规则过于宽泛。(5)引入的唯一实质性要求是存在一个非空的正面类比。显然，有些类比论证满足这个条件，但对其结论既没有初步的合理性，也没有支持的度量。

这里有一个简单的例子。阿钦斯坦（1964: 328）观察到，如果我们将关系“具有相同颜色”对应于“全等于”，那么天鹅和线段之间存在一个形式上的类比。这两种关系都是自反的、对称的和传递的。然而，从这个类比中找到正面支持，以支持我们可能会发现两个或更多全等线段聚集在一起的想法，就是荒谬的，仅仅因为相同颜色的天鹅通常是成群结队的。正面类比在先验上已知与假设类比无关。在这种情况下，类比推理应该被完全拒绝。然而，规则(5)会错误地给予非零的支持程度。

为了概括这个困难：并非每个相似性都增加了结论的概率，也并非每个差异都减少了它。有些相似性和差异被认为是完全无关紧要的，不应该对我们的概率判断产生任何影响。为了可行，规则（5）需要补充考虑“相关性”，这取决于主题、历史背景和每个类比论证的逻辑细节。因此，寻求一个简单的类比推理规则似乎是徒劳的。

Carnap和他的追随者（Carnap 1980; Kuipers 1988; Niiniluoto 1988; Maher 2000; Romeijn 2006）使用Carnapian的λγ规则为归纳逻辑制定了类比原则。一般来说，这个工作涉及到“相似性类比”，而不是这里讨论的类比推理类型。Romeijn（2006）认为Carnap的类比概念与类比预测之间存在关系。他的方法是Carnap式归纳规则和贝叶斯模型的混合体。这样的方法需要概括处理[§2.1](https://plato.stanford.edu/entries/reasoning-analogy/#Exa)中描述的类比论证类型。目前尚不清楚Carnap式方法能否提供类比推理的通用规则。

Norton（2010年和2018年——请参见其他互联网资源）认为，将归纳推理的项目形式化为一个或多个简单的形式图式是注定要失败的。当应用于类比推理时，他的批评尤为恰当。他写道：

> 如果类比推理只需要符合简单的形式图式，那么限制就太宽松了。将授权做出那些明显不该通过的推理…… 回应可以说是发展更复杂的形式模板…… 已知的困难是，这些装饰的图式似乎永远不够装饰；在分析中总会有一部分需要凭直觉从严格的形式规则中获得指导。（2018：1）

诺顿进一步提出了他的“材料理论”归纳推理的观点。他认为，没有一个普遍的逻辑原则可以“通过断言共享某些属性的事物必须共享其他属性”来“推动”类比推理。相反，每个类比推理都是由他称之为“类比事实”的目标系统的一些局部事实所证明的。这些局部事实应该根据具体情况进行确定和调查。

在类比问题上，完全采用形式化方法和完全摒弃形式化方法是两个极端。存在中间立场。最近的分析（无论是哲学的还是计算的）都致力于阐明类比推理的标准和程序，而不是形式规则。只要这些标准不打算提供一个普遍的“逻辑”类比，即使接受诺顿的基本观点，也还有空间来讨论这些标准。下一节将讨论其中一些标准和程序。

## 3. 评估类比论证的标准

### 3.1 常识指导原则

逻辑学家和科学哲学家已经确定了评估类比论证的“教科书式”一般准则（Mill 1843/1930; Keynes 1921; Robinson 1930; Stebbing 1933; Copi and Cohen 2005; Moore and Parker 1998; Woods, Irvine, and Walton 2004）。以下是其中一些最重要的准则：

 （G1）

越相似（两个领域之间），类比关系就越强大。

 （G2）

越是差异越大，类比的力量就越弱。

(G3)

我们对这两个领域的无知程度越大，类比的力量就越弱。

 （G4）

结论越弱，类比越有可能。

涉及因果关系的类比比不涉及因果关系的类比更有说服力。

 （G6）

结构类比比基于表面相似性的类比更强大。

 （G7）

相似性和差异对结论的相关性（即对假设类比的相关性）必须考虑在内。

 （G8）

多个支持同一结论的类比使得论证更加有力。

这些原则可能有所帮助，但通常过于模糊，无法提供太多的洞察力。在应用（G1）和（G2）时，我们如何计算相似性和差异性？为什么在（G5）和（G6）中提到的结构和因果类比特别重要，哪些结构和因果特征值得关注？更一般地，在与至关重要的（G7）相关的问题上：我们如何确定哪些相似性和差异性与结论相关？此外，对于在类比论证中被省略但仍可能相关的相似性和差异性，我们应该如何评价？

另一个问题是标准可能会朝不同的方向引导。举个例子，考虑里德关于其他星球上存在生命的论证（[*例子2*](https://plato.stanford.edu/entries/reasoning-analogy/#Example2)）。斯特宾（1933年）认为里德的论证“引人思考”且“不无道理”，因为结论较为薄弱（G4），而米尔（1843/1930年）似乎因为我们对可能相关的属性了解甚少而拒绝了这个论证（G3）。

还有一个问题与刚才提到的区别有关（在[§2.3](https://plato.stanford.edu/entries/reasoning-analogy/#Pla)中）。除了（G7）之外，上述每个标准都是以论证的强度来表达的，即对结论的支持程度。因此，这些标准似乎预设了对合理性的*概率*解释。问题在于，很多类比论证旨在建立*表面上看来*合理性，而不是任何概率程度。大多数准则对这类论证并不直接适用。

### 3.2 亚里士多德的理论

亚里士多德为后来所有关于类比推理的理论奠定了基础。在他对类比的理论反思和他最明智的例子中，我们找到了一个冷静的描述，为上述常识指导方针和更复杂的分析奠定了基础。

尽管亚里士多德使用了"类比"（*analogia*）这个术语，并讨论了[类比断言](https://plato.stanford.edu/entries/analogy-medieval/)，但他从未谈论过类比推理或类比论证本身。然而，他确实确定了两种论证形式，即"例证论证"（*paradeigma*）和"类似论证"（*homoiotes*），这两种形式与我们现在所认识的类比论证密切相关。

"例证论证"（*paradeigma*）在《修辞学》和《先验分析》中有描述：

基于例子的概括推理是指从一个或多个类似情况出发，得出一个普遍命题，然后通过演绎推理得出一个特定的推论。（《修辞学》1402b15） 假设A是邪恶的，B与邻国交战，C雅典人与底比斯人交战，D底比斯人与福基亚人交战。如果我们想要证明与底比斯人交战是邪恶的，我们必须假设与邻国交战是邪恶的。这个结论可以从类似的情况中得出，例如与福基亚人的战争对底比斯人来说是邪恶的。因此，既然与邻国交战是邪恶的，而与底比斯人交战就是与邻国交战，那么与底比斯人交战就显然是邪恶的。（《分析前书》69a1）

亚里士多德指出了这种论证形式与归纳推理的两个不同之处（69a15ff.）：它“并不从所有的特例中得出证明”（即它不是“完全”的归纳），并且它需要一个额外的（演绎有效的）三段论作为最后一步。例证论证因此相当于单个案例的归纳，然后是演绎推理。它具有以下结构（使用⊃表示条件）：

在 [§2.2](https://plato.stanford.edu/entries/reasoning-analogy/#Cha) 的术语中，P 是正面类比，Q 是假设类比。在亚里士多德的例子中，S（源）是福基亚人和底比斯人之间的战争，T（目标）是雅典人和底比斯人之间的战争，P 是邻居之间的战争，Q 是邪恶。第一个推理（虚线箭头）是归纳的；第二个和第三个推理（实线箭头）是演绎有效的。

这个 *paradeigma* 有一个有趣的特点：它可以作为一个纯粹的 *演绎* 论证形式进行替代分析。让我们专注于亚里士多德的断言，“我们必须假设与邻居作战是邪恶的”，表示为 ∀x(P(x)⊃Q(x))。我们可以将这个中间步骤视为从单个案例归纳得出的东西，也可以将其视为一个隐藏的前提。这将 *paradeigma* 转化为一个带有缺失或 *enthymematic* 前提的三段论论证，我们的注意力转向建立该前提的可能手段（其中单例归纳是其中一种手段）。以这种方式解释，亚里士多德的 *paradeigma* 论证预示了类比推理的演绎分析（参见 [§4.1](https://plato.stanford.edu/entries/reasoning-analogy/#DedJus)）。

*类比论证*（*homoiotes*）似乎比*范例*（*paradeigma*）更接近我们对类比论证的当代理解。这种论证形式在《论题》第一卷的第17和18章以及第八卷的第1章中受到了相当大的关注。最重要的段落如下。

> 尝试通过类似性来获得承认；因为这样的承认是合理的，而所涉及的普遍性不太明显；例如，知识和无知是相同的，对立面的感知也是相同的；或者反过来，感知相同，知识也是相同的。这种论证类似归纳，但并不完全相同；因为在归纳中，是从个别事例中获得了普遍性的承认，而在类比论证中，获得的并不是所有相似情况都适用的普遍性。（《论题》156b10–17）

这段文字出现在一篇提供如何在面对有些怀疑的对话者时构建辩证论证的作品中。在这种情况下，最好不要让自己的论点依赖于对任何普遍命题的达成一致。因此，类比论证与范例论证明显不同，范例论证中普遍命题在论证中起到了必要的中间步骤的作用。类比论证虽然逻辑上比范例论证更加复杂，但当我们对基本概括不确定时，它恰好是我们想要的那种类比推理。

在《论题》I 17中，亚里士多德指出任何共享属性都会产生一定程度的相似性。自然而然地会问，两个事物之间的相似程度何时足够大，以证明进一步的相似性。换句话说，类比论证何时成功？亚里士多德没有明确回答，但通过他对特定类比论证的证明方式，我们可以找到一些线索。正如劳埃德（1966）所观察到的，亚里士多德通常通过阐明统治两个现象的（有时模糊的）因果原则来证明这类论证。例如，亚里士多德通过将海水的咸味类比为汗水的咸味，解释了海水的咸味是一种在加热等自然过程中排出的残留土质物质。这个共同原则就是：

一切生长和自然生成的事物总是留下残留物，就像燃烧后的物体一样，其中包含这种土壤。 （《天文学》358a17）

从这种证明方法中，我们可以推测亚里士多德认为重要的相似之处是那些涉及到这种一般因果原则的。

总结一下，亚里士多德的理论为我们提供了四个重要且有影响力的评估类比论证的标准：

- *类比的强度取决于相似性的数量。* - *相似性归结为相同的属性和关系。* - *好的类比源于共同的根本原因或普遍规律。* - *一个好的类比论证不需要预设对底层普遍规律的了解。*

这四个原则构成了一个用于评估类比论证的“常识模型”的核心（这并不意味着它们是正确的；事实上，前三个原则很快就会受到质疑）。第一个原则，正如我们所见，经常出现在类比的教科书讨论中。第二个原则在很大程度上被默认为正确，但在计算模型的类比中有重要的例外（§3.4）。第三个原则的版本在大多数复杂理论中都能找到。最后一个观点将这个论证与相似性论证和例证论证区分开来，在许多类比的讨论中得到认可（例如，奎因和乌利安，1970年）。

对亚里士多德第一个原则的轻微概括有助于为后来的发展讨论做准备。正如该原则所暗示的，亚里士多德与几乎所有写过类比推理的人一样，将他对论证形式的分析组织在整体相似性的基础上。在“水平关系”的术语中，推理的驱动力是“两个领域的整体相似性越大，类比论证越强”。休谟在他的《论自然宗教的对话》中也提到了同样的观点，尽管是以否定的方式陈述的。

无论你离开相似案例的程度有多小，你都会相应地减少证据；最终可能会得出一个非常薄弱的类比，被公认为容易出错和不确定的。（1779/1947: 144）

大多数类比理论都同意亚里士多德和休谟的这个总体观点。分歧在于衡量整体相似性的适当方式。一些理论赋予最大的重视*物质类比*，即共享的、通常可观察的特征。其他理论则突出强调*形式类比*，强调高层次的结构对应。接下来的两个小节讨论了代表性的观点，说明了这两种方法。

### 3.3 材料标准：赫尔米亚的阿莫尼俄斯理论

赫尔米亚的阿莫尼俄斯（Hesse，1966）提出了亚里士多德理论的一个更加精炼的版本，特别关注科学中的类比论证。她提出了三个要求，以确保一个类比论证是可接受的：

1. *物质类比的要求*。水平关系必须包括可观察属性之间的相似性。2. *因果条件*。垂直关系必须是“在某种可接受的科学意义上的”因果关系（1966年：87）。3. *无本质差异条件*。源领域的基本属性和因果关系不能被证明是负类比的一部分。

#### 3.3.1 物质类比的要求

对于赫尔米亚来说，一个可接受的类比论证必须包括不同领域之间的“可观察相似性”，她称之为“物质类比”。物质类比与“形式类比”相对。如果两个领域都是“同一形式理论的解释”，则它们在形式上是类似的（1966: 68）。*规范同构*（Hempel 1965）是一种特殊情况，其中控制两个系统的物理定律具有相同的数学形式。热量和流体流动表现出规范同构。第二个例子是导线中电流流动和管道中流体流动之间的类比。欧姆定律

相比之下，物质类比由Hesse称为“可观察的”或“前理论的”相似之处组成。这些是源和目标对象的属性之间的相似性的水平关系。例如，我们在对这些现象没有任何详细理论之前就已经认识到了回声（声音）和反射（光）之间的相似性。Hesse（1966, 1988）将这种相似性视为两个领域之间的隐喻关系，并将其标记为“前理论的”，因为它们依赖于个人和文化经验。我们在声音和光之间既有物质类比又有形式类比，对于Hesse来说，前者独立于后者是很重要的。

有很好的理由不接受赫尔米亚的要求，以这种狭义的方式解释物质类比。首先，显然在许多重要的推理中，形式类比是起点。这在数学中无疑是如此，而在赫尔米亚的意义上，物质类比根本不起作用。基于形式类比的类比论证在物理学中也具有极大的影响力（Steiner 1989, 1998）。

然而，在诺顿的广义上，“物质类比”只是指源领域和目标领域的实际知识根源上的相似之处。在这个更广义的意义上，赫尔米亚提出了两个额外的物质标准。

#### 3.3.2 因果条件

赫尔米亚的阿莫尼俄斯要求假设的类比，即转移到目标领域的特征，与正面类比之间存在*因果关系*。她认为，从类比中得出一个好的论证的基本要求是“共同出现的倾向”，即因果关系。她将这一要求表述如下：

模型中的垂直关系是在某种可接受的科学意义上的因果关系，没有令人信服的先验理由来否认解释对象之间可能存在相同类型的因果关系。（1966年：87）

因果条件排除了在源领域没有因果知识的情况下进行类比论证的可能性。它得到了这样的观察支持，即许多类比似乎涉及到因果知识的转移。

因果条件是正确的，但可以说过于限制性。例如，它排除了数学中的类比论证。即使我们将注意力限制在经验科学上，有说服力的类比论证可能建立在强大的统计相关性基础上，而无需任何已知的因果关系。考虑（示例11）本杰明·富兰克林在1749年的预测，即尖金属棒会像实验室中吸引“电流”一样吸引闪电：

> 电流在以下几个方面与闪电相符：1. 发光。2. 光的颜色。3. 弯曲的方向。4. 快速运动。5. 被金属导电。6. 爆炸时发出裂缝或噪音。7. 在水或冰中存在。8. 穿过物体时撕裂。9. 毁灭动物。10. 熔化金属。11. 点燃易燃物质。12. 有硫磺味道。- 电流被尖端吸引。- 我们不知道闪电是否具有这种特性。- 但既然它们在我们已经可以比较的所有细节上都相符，它们是否也可能在这一点上相符呢？让我们进行实验吧。（《本杰明·富兰克林的实验》，334页）

弗兰克林的假设基于目标（闪电）和源（实验室中的电流）之间的一长串共同特性。这十二个“特定事项”与第十三个特性之间没有已知的因果关系，但存在着强烈的相关性。即使没有已知的因果关系，类比论证也可能是合理的。

#### 3.3.3 无本质差异条件

赫尔米亚的阿莫尼俄斯的最后要求是，“[源]的基本属性和因果关系尚未被证明是负面类比的一部分”（1966年：91）。赫尔米亚的阿莫尼俄斯没有提供“基本”的定义，但暗示如果某个属性或关系与已知的正面类比“因果密切相关”，则它是基本的。例如，流体流动的类比在发展导热理论方面具有极大影响力。然而，一旦发现热量不守恒，这个类比就变得不可接受（根据赫尔米亚的阿莫尼俄斯），因为守恒对于流体流动理论非常重要。

尽管这个要求再次朝着正确的方向，但似乎过于限制性。它可能导致拒绝一个好的类比论证。考虑二维矩形和三维盒子之间的类比（[*例子7*](https://plato.stanford.edu/entries/reasoning-analogy/#Example7)）。扩大赫尔米亚的阿莫尼俄斯的概念，似乎矩形和盒子之间有许多“基本”的差异。这并不意味着我们应该一概而论地拒绝矩形和盒子之间的每一个类比。问题在于赫尔米亚的阿莫尼俄斯的条件是独立于该关系的使用而应用于类比的关系。什么被视为基本应该随着类比论证的不同而变化。在没有推理上下文的情况下，评估相似性和差异的重要性或“基本性”是不可能的。

尽管存在这些弱点，赫尔米亚的阿莫尼俄斯的“物质”标准在我们理解类比推理方面取得了重要进展。因果条件和无本质差异条件将诺顿所提倡的局部因素纳入类比论证的评估中。这些条件单独或合并使用，意味着即使存在非空的积极类比，类比论证也可能无法为其结论提供任何支持。赫尔米亚的阿莫尼俄斯没有提供关于类比支持“程度”的理论。这使得她的观点成为少数几个面向模态而非概率使用类比论证的观点之一（[§2.3](https://plato.stanford.edu/entries/reasoning-analogy/#Pla)）。

### 3.4 形式标准：结构映射理论

许多人将“模型论同构”概念作为思考相似性及其在类比推理中的作用的标准。他们提出了基于整体结构或句法相似性的“形式标准”来评估类比。让我们将以这些标准为导向的理论称为“结构主义”。

许多领先的类比计算模型都是结构主义的。它们通过计算机程序实现，该程序从源领域和目标领域开始（或有时构建）表示，然后构建可能的类比映射。类比推理是通过确定“最佳映射”而产生的。就类比推理的标准而言，有两个主要观点。首先，一个“类比论证”的好坏取决于相关的“类比映射”的好坏。其次，类比映射的好坏由一个指示其近似同构程度的度量标准给出。

最有影响力的结构主义理论是赫尔米亚的阿莫尼俄斯的*结构映射*理论，该理论在一个名为*结构映射引擎*（SME）的程序中得到实现。在其最初的形式中（Gentner 1983），该理论仅基于结构性的考量来评估类比。赫尔米亚阐述道：

> 类比是关于关系而非简单特征的。无论是什么类型的知识（因果模型、计划、故事等），都是结构性属性（即事实之间的相互关系）决定了类比的内容。（Falkenhainer, Forbus, and Gentner 1989/90: 3）

为了阐明这个论点，根特纳引入了一个区分*属性*或一元谓词和*关系*的概念，后者具有多个参数。她进一步区分了不同的*关系顺序*和*函数顺序*，这些顺序是归纳定义的（根据关系或参数的顺序）。最佳映射由*系统性*决定：它将高阶关系和嵌套在高阶关系中的项目放置在对应位置的程度。根特纳的*系统性原则*如下所述：

> 属于可映射的相互连接关系系统的谓词比孤立的谓词更有可能被引入目标中。（1983年：163）

系统的类比（将高阶关系及其组成部分对应起来的类比）比不太系统的类比更好。因此，类比推理的合理性程度随相关类比映射的系统性程度单调增加。Gentner对评估候选类比（和类比推理）的基本标准仅依赖于给定表示的语法，而不依赖于其内容。

结构映射理论的后续版本包括一些改进（Forbus、Ferguson和Gentner 1994；Forbus 2001；Forbus等人2007；Forbus等人2008；Forbus等人2017）。例如，该理论的最早版本容易受到关于源领域和目标领域手工编码表示的担忧。Gentner及其同事们在后续工作中尝试解决这个问题，通过自然语言文本生成LISP表示（参见Turney 2008提供的不同方法）。

结构映射方法面临的最重要挑战与“系统性原则”本身有关。类比的价值是否完全或主要来自系统性？这种观点似乎存在两个主要困难。首先，将优先考虑系统性的高级关系匹配并不总是合适的。在某些类型的类比推理中，材料标准，尤其是Gentner所称的“表面特征匹配”，可能非常重要，例如基于物件之间表面相似性的民族志类比。其次，更重要的是，系统性似乎最多只能作为好类比的一个不可靠的“标记”，而不是好类比推理的本质。

更大的系统性既不是更合理类比推理的必要条件，也不是充分条件。显然，增加系统性并不足以增加合理性。一个不合理的类比可以以一种展示高度结构平行性的形式来表示。高阶关系可以很容易地获得，正如我们在Achinstein的“天鹅”例子中所看到的。

更具体地说，增加系统性并不是增加可信度的 *必要* 条件。实际上，在因果类比中，它甚至可能削弱推断。这是因为 *系统性* 并不考虑因果相关性，无论是正面还是负面。 （McKay 1993）指出，在南极冰湖中发现了微生物；类比，火星上可能存在简单的生命形式。 冰冷的温度是 *预防性* 或对抗性原因；它们与生命的存在是 *负面相关* 的。 35亿年前，火星的气候可能比今天更有利于生命，因为温度较暖。然而，南极与今天的火星之间的类比比南极与古代火星之间的类比更为 *系统化*。 根据 *系统性原则*，与南极的类比为今天的火星上的生命提供了比古代火星上的生命更强的支持。

这个例子的要点是，增加系统性并不总是增加可信度，减少系统性也不总是降低可信度（请参见Lee和Holyoak 2008）。更普遍的观点是，系统性可能具有误导性，除非我们考虑各种因素之间的关系性质和假想类比之间的关系。系统性并不能神奇地产生或解释类比论点的可信度。当我们通过类比推理时，我们必须确定两个领域的哪些特征是相关的，以及它们与类比结论的关系* 如何* 。没有语法可以捷径。

Schlimm（2008）从数学的类比推理的角度，对结构映射理论提出了完全不同的批评，数学是一个人们可能期望像*结构映射*这样的形式方法表现良好的领域。Schlimm提出了一个简单的区别：如果对象的数量大于关系（和属性）的数量，则该领域是*对象丰富*的，否则是*关系丰富*的。结构映射理论的支持者通常关注关系丰富的例子（例如太阳系和原子之间的类比）。相比之下，数学中的类比通常涉及具有大量对象（如实数）但相对较少的关系和函数（加法，乘法，小于）的领域。

Schlimm提供了一个涉及每个领域中单个关系的群论类比推理问题的例子。在这种情况下，达到最大系统性是微不足道的。困难在于，与最大系统性相兼容的是，对象可能以不同的方式进行对应。结构映射理论似乎得出了错误的推论。我们可以总结为：在对象丰富的领域中，系统性不再是可靠的合理类比推理指南。

 ### 3.5 其他理论

#### 3.5.1 连接主义模型

在过去的三十五年里，认知科学家对类比进行了广泛的研究。赫尔曼的SME只是许多计算理论之一，它被实现在构建和使用类比的程序中。涵盖了这一时期的三本有帮助的文集是Helman 1988；Gentner, Holyoak, and Kokinov 2001；以及Kokinov, Holyoak, and Gentner 2009。

这项研究的一个主要目标是模拟使用类比所涉及的认知过程。早期的模型倾向于“理解规定人类类比思维的基本约束”（Hummel and Holyoak 1997: 458）。最近的连接主义模型则致力于揭示我们使用类比时涉及的心理机制：检索相关的源领域，跨领域进行类比映射，信息的转移以及新类别或模式的学习。

在某些情况下，比如结构映射理论（§3.4），这项研究直接与本文的焦点问题重叠；事实上，根特纳的《系统性原则》可以被解释为规范性的。在其他情况下，我们可以将这些项目视为用最新的计算形式的自然化认识论来替代传统的规范性问题。这里特别提到了两种方法，因为它们都对找到这些问题的明确答案的想法提出了重要的挑战，并且都表明连接主义模型提供了一种更有成效的理解类比推理的方法。

第一种是约束满足模型（也称为多约束理论），由霍利奥克和塔加德（1989年，1995年）开发。与根特纳一样，霍利奥克和塔加德认为类比推理的核心是类比映射，并强调系统性的重要性，他们将其称为一种结构性约束。与根特纳不同的是，他们承认还有两种额外的约束类型。实用约束考虑了代理人的目标和目的，认识到“目的将指导选择”相关的相似之处。语义约束代表了人们将源项和目标项视为相似程度的估计，有点类似赫斯的“前理论”相似之处。

多重约束理论的新颖之处在于这些*结构*、*语义*和*语用*约束并非作为严格的规则实施，而是作为支持或抑制潜在的成对对应关系的“压力”。该理论在一个名为**ACME**（类比约束映射引擎）的连接主义程序中实现，该程序为源域和目标域中的每个可能的配对元素分配一个初始激活值（基于语义和语用约束），然后通过循环更新激活值，根据整体一致性（结构约束）。在这些约束的压力下，最佳的全局类比映射出现。随后的连接主义模型，如Hummel和Holyoak的LISA程序（1997年，2003年），取得了重大进展，并有望提供更完整的类比推理理论。

第二个例子是Hofstadter和Mitchell的***Copycat***程序（Hofstadter 1995; Mitchell 1993）。该程序“旨在以心理学上逼真的方式发现有洞察力的类比”（Hofstadter 1995: 205）。Copycat在字母字符串领域操作。该程序处理以下类型的问题：

假设字母串 *abc* 被改变为 *abd*；你会如何以“相同的方式”改变字母串 *ijk*？

大多数人会回答 *ijl*，因为自然而然地认为 *abc* 被改变为 *abd* 是通过“转换规则”：将最右边的字母替换为它的后继字母。也可能有其他的答案，但与大多数人对于什么算作自然类比的感觉不一致。

Hofstadter和Mitchell认为，类比思维在很大程度上涉及对新颖模式的*感知*，而这种感知需要具有“流动”边界的概念。真正的类比思维涉及概念的“滑动”。Copycat程序将与字母序列相关的一组核心概念（*后继*、*最左边*等）与动态链接不同概念的概率“光环”相结合。有序结构从随机的低层过程中出现，程序产生了合理的解决方案。因此，Copycat表明，类比思维可以被建模为一种类似于感知的过程，即使该程序使用的机制与人类感知中的机制不同。

多约束理论和Copycat共享这样一个观点，即类比认知涉及在抽象推理水平之下运作的认知过程。两个计算模型——在能够进行成功的类比推理的程度上——挑战了一个成功的类比推理模型必须采取一组准逻辑标准的观点。可以说，发展一种准逻辑的*理论*类比推理的努力已经失败。计算模型取而代之的是那些可以根据其性能而不是传统哲学标准来评判的*过程*。

针对这个论点，我们应该承认连接主义模型的价值，同时也要认识到我们仍然需要一个理论来提供评估类比论证的规范原则。首先，即使类比的构建和识别在很大程度上是感知的问题，这并不排除对类比推理进行后续批判性评估的需要。其次，更重要的是，我们需要关注的不仅仅是类比映射的构建，还包括数学、物理、哲学和法律等领域中个别类比论证的辩论方式。这些高层次的辩论需要的推理与ACME或Copycat的计算过程几乎没有任何相似之处。（Ashley的HYPO（Ashley 1990）是一个关注类比推理这一方面的非连接主义程序的例子。）因此，计算模型和传统哲学模型在类比推理方面都有存在的空间。

#### 3.5.2 表达模型

大多数关于类比的突出理论，无论是哲学的还是计算的，都是基于源领域和目标领域之间的整体相似性，这些相似性是以赫斯的“水平关系”的某个偏爱子集来定义的（参见[§2.2](https://plato.stanford.edu/entries/reasoning-analogy/#Cha)）。亚里士多德和米尔的方法在教科书讨论中有所体现，他们建议计算相似性。赫斯的理论（[§3.3](https://plato.stanford.edu/entries/reasoning-analogy/#MatCriHesThe)）偏爱“预理论”的对应关系。结构映射理论及其后继理论（[§3.4](https://plato.stanford.edu/entries/reasoning-analogy/#ForCriStrMapThe)）则关注系统性，即涉及复杂的、高层次的关系网络的对应关系。在这些方法中，问题是双重的：整体相似性不是一个可靠的指导可信度的方法，而且它无法解释任何类比论证的可信度。

巴尔塔的“表达模型”（2010）提出了一种不同的方法，它不是从水平关系开始，而是根据每个领域内的“垂直关系”对类比论证进行分类。其基本思想是，一个好的类比论证必须满足两个条件：

*先前的关联*。在源领域中，已知的相似之间必须有明确的联系（正向类比），并且进一步推测在目标领域中也存在相似之处（假设类比）。这种关系决定了源领域中哪些特征对于类比推理是*关键*的。

*潜在的普遍性*。必须有理由认为在目标领域中可能存在相同类型的联系。更明确地说：领域之间不能存在*关键的*不类比之处。

首要任务是明确先前的关联。明确性的标准取决于这种关联的性质（因果关系、数学证明、功能关系等）。这两个一般原则通过一组从属模型得到充实，这些模型使我们能够识别关键特征，从而识别关键的不类比之处。

为了看清楚这是如何运作的，请考虑[*示例7*](https://plato.stanford.edu/entries/reasoning-analogy/#Example7)（矩形和盒子）。在这个类比论证中，源领域是二维几何：我们知道，在具有固定周长的所有矩形中，正方形具有最大面积。目标领域是三维几何：类比地，我们猜测，在具有固定表面积的所有盒子中，立方体具有最大体积。这个论证应该通过计算相似性的数量、寻找矩形和盒子之间的前理论相似之处，或者构建领域的连接主义表示并计算可能映射的系统性得分来评估。相反，我们应该从源领域中对先前关联进行精确表述开始，这相当于对矩形结果的具体证明。然后，我们应该相对于该证明，确定源领域的关键特征，即证明中使用的概念和假设。最后，我们应该评估泛化的潜力：在三维环境中，这些关键特征是否已知在目标领域中缺乏类似物。表述模型旨在反映在类比论证的支持者和批评者之间可能发生的对话。

### 3.6 基于实践的方法

基于科学实践的类比推理研究为评价类比论证提供了有价值的观点。

#### 3.6.1 诺顿的类比物质理论

正如在[§2.4](https://plato.stanford.edu/entries/reasoning-analogy/#AnaInfRul)中所指出的，诺顿拒绝了类比推理规则。但即使我们在这一点上同意诺顿的观点，我们可能仍然对拥有一个能够为我们评估类比论证提供指导的解释感兴趣。诺顿的方法在这方面表现如何？

根据诺顿的观点，每个类比论证都是由必须经验调查和证明的局部事实支持的。首先，有“类比事实”：在实践中，这是一种包括源系统和目标系统的低级统一性。其次，目标系统的附加事实属性与统一性一起，支持类比推理。考虑伽利略的著名推理（***例子12***），即月球上有山脉（伽利略1610）。通过他新发明的望远镜，伽利略观察到月球上的光点在阳光前沿之前出现。他注意到当阳光照射到山脉时，地球上也会发生同样的情况，因此他得出结论月球上一定有山脉，并且还提供了它们的合理高度估计。诺顿告诉我们，在这个例子中，“类比事实”是阴影和其他光学现象在地球和月球上以相同的方式产生；关于目标的附加事实是月球上阳光前沿之前的光点的存在。

诺顿的物质理论对评估类比论证有什么影响？“类比事实”是推理的一种局部统一性。当这种统一性是明显的或自然推断出来时，诺顿的理论效果很好。但当这种统一性本身成为推理的“目标”（而不是“驱动力”）时，效果就不好了。这种情况发生在解释性类比中，例如[*例子5*]（https://plato.stanford.edu/entries/reasoning-analogy/#Example5）（声学类比），以及数学类比，例如[*例子7*]（https://plato.stanford.edu/entries/reasoning-analogy/#Example7）（矩形和盒子）。同样，当基础统一性不明确时，理论也不起作用，例如[*例子2*]（https://plato.stanford.edu/entries/reasoning-analogy/#Example2）（其他星球上的生命），[*例子4*]（https://plato.stanford.edu/entries/reasoning-analogy/#Example4）（陶罐）以及许多其他情况。简而言之，如果接受诺顿的理论，那么对于大多数类比论证，没有有用的评估标准。

#### 3.6.2 领域特定的标准

对于那些对诺顿对普遍归纳方案和类比推理理论的怀疑持同情态度，但又认识到他的方法可能过于局限，一种吸引人的策略是向上提升一个层次。我们可以追求领域特定的“工作逻辑”（Toulmin 1958；Wylie和Chapman 2016；Reiss 2015）。这种方法已经被考古学家、进化生物学家和其他历史科学家采用（Wylie和Chapman 2016；Currie 2013；Currie 2016；Currie 2018）。在模式的位置上，我们找到了“工具包”，即用于评估类比推理的标准列表。

例如，Currie（2016）详细探讨了民族志类比（***示例13***）的使用，即当代桑人使用的萨满主义图案与古代岩石艺术中的类似图案之间的类比，这些图案既可以在桑人的祖先中找到（直接历史类比），也可以在欧洲岩石艺术中找到（间接历史类比）。类比论证支持这样的假设，即在这些文化中，岩石艺术象征着幻觉体验。Currie考察了关于文化特征的稳定性和环境-文化关系的假设的标准。Currie（2016，2018）和Wylie（Wylie和Chapman 2016）还强调了将中等强度的类比论证与其他形式的证据相结合以得出强有力结论的鲁棒性推理的重要性。

因此，基于实践的方法可以提供特定的指导方针，这是任何一般的类比推理理论所无法匹配的。值得一提的是，有一个需要注意的问题。民族志类比的领域特定标准是在数十年的方法论争议的背景下引出的（Wylie和Chapman 2016）。对于民族志类比的批评者和捍卫者，他们曾经诉诸于科学方法的一般模型（例如，假设演绎法或贝叶斯确认）。为了推进方法论的辩论，基于实践的方法必须要么与这些一般模型建立联系，要么解释为什么缺乏这种联系是无问题的。

#### 3.6.3 物理学中的形式类比

对实践中的类比论证给予密切关注，也可以对关于类比推理的一般观念提供有价值的挑战。在一次有趣的讨论中，斯坦纳（Steiner）（1989年，1998年）提出，20世纪初物理学中发挥重要作用的许多类比都可以被称为“毕达哥拉斯式”。这个术语意味着数学神秘主义：一个“毕达哥拉斯式”类比是一种纯粹的形式类比，它建立在数学相似性的基础上，而这些相似性在提出时没有已知的物理基础。一个例子是薛定谔（Schrödinger）使用类比（***例子14***）来“猜测”相对论波动方程的形式。在斯坦纳的观点中，薛定谔的推理依赖于纯粹数学类比的操作和替换。斯坦纳认为，这种类比的成功，甚至合理性，“引发了或应该引发困惑”（1989年：454）。赫斯（Hesse）（1966年）和巴尔塔（Bartha）（2010年）都反对这样一个观点：在物理学中，一个纯粹的形式类比，没有物理意义，不能支持一个合理的类比推理。因此，斯坦纳的论点提出了一个严肃的挑战。

Bartha（2010）提出了一个回应：我们可以将Steiner的例子分解为两个或更多步骤，然后确定至少有一步确实有物理基础。然而，Fraser（即将出版）提供了一个支持Steiner观点的反例。经典统计力学（CSM）和量子场论（QFT）之间的复杂类比在两个理论中的重整化群（RG）方法的发展和应用中起到了关键作用（***例子15***）。Fraser指出了CSM和QFT之间的实质性物理不相似之处，并得出结论认为这种推理完全基于形式类比。

## 4.类比推理的哲学基础

哲学基础可以为类比推理提供什么样的依据？对于类比论证能够得出合理结论的主张，可以给出什么样的理由？对于回答这个问题，已经有几种想法。一种自然的策略是将类比推理与其他已经理解的论证模式相结合，即演绎或归纳推理的形式。一些哲学家探讨了先验的合理性可能性。实际应用类比可能存在实用的合理性，尤其是在法律推理中。

任何试图为类比推理提供普遍合理性的尝试都面临着一个基本的困境。普遍性的要求需要对问题进行高层次的表述，因此需要对类比论证进行抽象的描述，例如模式（4）。另一方面，正如前面所提到的，许多符合模式（4）的类比论证都是不好的论证。因此，类比推理的普遍合理性不能为所有符合（4）的论证提供支持，否则就会证明过多。相反，它必须首先指定一组被认为是“好”的类比论证的子集，并将普遍合理性与这个指定的子集联系起来。合理化问题与描述好的类比论证问题相关。这个困难影响了本节中描述的一些策略。

### 4.1 演绎的证明

类比推理可以被视为一种“演绎”的模式。如果成功，这种策略可以很好地解决证明的问题。一个有效的演绎论证是最好的。

早期的演绎主义方法的一个例子是亚里士多德对于例证论证（paradeigma）的处理。根据这种分析，源领域S和目标领域T之间的类比论证始于对于正面类比P(S)和P(T)的假设，以及额外的信息Q(S)。通过普遍化∀x(P(x)⊃Q(x))，推导出结论：Q(T)。只要我们能将这个中间的普遍化视为独立前提，我们就有了一个演绎有效的论证。然而需要注意的是，普遍化的存在使得类比变得无关紧要。我们可以从普遍化和P(T)推导出Q(T)，而不需要对源领域有任何了解。关于论证中的类比的文献还提供了更多关于这种分析的观点，以及关于类比论证是否适当地被描述为演绎的问题。

一些最近的分析遵循亚里士多德的观点，将类比论证视为依赖于额外的（有时是隐含的）前提的，这些前提通常来自背景知识，将推理转化为演绎有效的论证，但并不使源领域变得无关紧要。戴维斯和拉塞尔引入了一种依赖于他们所称的“确定规则”的版本（Russell 1986; Davies and Russell 1987; Davies 1988）。假设Q和P1,...,Pm是变量，并且我们有背景知识，即Q的值由P1,...,Pm的值决定。在最简单的情况下，当m=1且P和Q都是二进制布尔变量时，这可以简化为

(8)∀x(P(x)⊃Q(x))∨∀x(P(x)⊃∼Q(x))，

即，P是否成立决定了Q是否成立。更一般地，决定规则的形式是

 （9）Q=F（P1，…，Pm），

即，Q是P1，…，Pm的函数。如果我们假设这样的规则是我们的背景知识的一部分，那么具有结论Q（T）的类比论证是演绎有效的。更准确地说，允许Q不是二进制变量的情况：如果我们有这样的规则，并且还有前提陈述源S与目标T在所有值Pi上一致，那么我们可以有效地推断出Q（T）=Q（S）。

“决定规则”分析为类比推理提供了一个清晰而简单的理由。请注意，与亚里士多德通过普遍化∀x(P(x)⊃Q(x))的分析相比，决定规则并不使类比论证变得琐碎。只有通过将规则与源领域的信息相结合，我们才能推导出Q(T)的值。为了说明这一点，我们可以借用罗素和戴维斯（***例子16***）给出的例子之一，假设一个二手车（相对于特定买家）的价值(Q)由其年份、制造商、里程、状况、颜色和事故历史（变量Pi）决定。这些因素中的一个或多个是否多余或无关紧要并不重要。只要两辆车在这些方面都无法区分，它们的价值就是相同的。了解源领域的知识是必要的；我们不能仅凭决定规则推导出第二辆车的价值。韦岑费尔德（1984）提出了这种方法的一个变体，提出了稍微更一般的论点，即类比论证是带有一个缺失的（潜在的）前提的演绎论证，该前提等同于一个决定规则。

决定规则是否为我们提供了解决类比论证的问题的方法？总的来说：不是。类比常常被应用于问题，例如[*例子8*](https://plato.stanford.edu/entries/reasoning-analogy/#Example8)（吗啡和哌替啶），在这些问题中，我们甚至不知道所有相关因素，更不用说拥有一个决定规则了。医学研究人员在动物身上进行药物测试时，并不知道可能与药物效果相关的所有属性。事实上，这类测试的主要目标之一就是防范理论未预料到的反应。根据“决定规则”分析，我们要么将这类论证的范围限制在我们拥有有力支持的决定规则的情况下，要么将注意力集中在制定和证明一个适当的决定规则上。对于动物测试等情况，两种选择似乎都不现实。

将类比重新构建为演绎论证可能有助于揭示背景假设，但对于证明问题来说并没有太大进展。这个问题重新出现为需要阐明和建立一个确定性规则的合理性，这至少和证明原始类比论证一样困难。

### 4.2 归纳证明

一些哲学家试图以某种被广泛理解的归纳论证模式来描绘和证明类比推理。这种策略有三个相对受欢迎的版本。第一个版本将类比推理视为从单个案例进行的概括。第二个版本将其视为一种抽样论证。第三个版本承认类比论证作为一种独特形式，但将过去的成功视为未来成功的证据。

#### 4.2.1 单例归纳

让我们重新考虑亚里士多德的例证论证或*paradeigma*（[§3.2](https://plato.stanford.edu/entries/reasoning-analogy/#AriThe)），但这次将归纳的普遍化视为从单个案例（源领域）归纳而来的合理化。这样简单的类比论证分析能成功吗？总的来说：不行。

有时候，一个单一的实例可以导致合理的普遍化。卡特赖特（1992）认为，我们有时可以从一个仔细的实验中进行普遍化，“在这个实验中，我们对材料有足够的控制，并且对所需的背景假设的知识是可靠的”（51页）。卡特赖特认为，我们可以在具有稳定的“亚里士多德性质”的化合物实验中做到这一点。在类似的精神中，奎因（1969）认为，在处理自然种类时，我们可以进行实例确认。

即使我们接受存在这样的情况，将所有类比论证理解为单例归纳的反对意见是显而易见的：这种观点过于限制性。大多数类比论证都不符合必要条件。当我们进行类比论证时，我们可能不知道我们正在处理的是自然种类还是亚里士多德的本质。我们可能不知道哪些属性是必要的。坚持将类比推理解释为“单例归纳”很可能导致怀疑（Agassi 1964, 1988）。

将类比论证解释为单例归纳在另一方面也是适得其反的。这种简单化的分析对于寻找帮助我们区分相关和无关相似之间的标准，从而区分好的和坏的类比论证毫无助益。

#### 4.2.2 采样论证

在类比论证的采样观念中，承认两个领域之间的相似之处被视为进一步相似性的统计相关证据。采样论证的最简单版本归功于米尔（1843/1930）。他写道，类比论证是“已知一致点和已知差异点之间的竞争。” A和B在10个属性中有9个一致意味着B具有A的任何其他属性的概率为0.9：“我们可以合理地期望相似性的比例相同”（367）。他唯一的限制与样本大小有关：我们必须对A和B都相对了解。米尔认为，利用类比推理来推断新发现的植物或动物物种的特征并没有困难，因为我们对植物学和动物学有广泛的知识。但是，如果A和B的未确定属性的范围很大，那么在小样本中的相似性将不是一个可靠的指南；因此，米尔对里德关于其他星球上的生命的论证的驳斥（[*例子2*](https://plato.stanford.edu/entries/reasoning-analogy/#Example2)）。

采样论证是由哈罗德（1956）以更明确的数学形式提出的。关键思想是，已知的S（源领域）的属性可以被视为所有S属性的随机样本，即相对于也属于T（目标领域）的属性而言是随机的。如果属于S的已知属性中大多数也属于T，那么我们应该期望S的大多数其他属性也属于T，因为我们不太可能只了解到共同属性。实际上，哈罗德提出了一个二项分布，模拟了从一个瓮中随机选择属性的过程。

哈罗德和米尔的分析存在严重困难。一个明显的困难是“计数问题”：属性的“总体”定义不清楚。我们如何计算相似性和差异性？根据我们如何做这个计算，共享属性与已知属性的总数之比会有很大的变化。第二个严重的困难是“偏见问题”：我们无法证明已知特征的样本是随机的。在瓮的情况下，选择过程被安排得不受代理人的意图、目的或先前选择的影响。相比之下，类比论证的呈现总是带有偏见的。偏见进入相似性和差异性的初始表达中：论证的支持者会强调相似性，而批评者会突出差异。从瓮中重复选择的范例似乎完全不合适。采样方法的其他变体已经被开发出来（例如，拉塞尔1988年），但最终这些版本也未能解决计数问题或偏见问题。

#### 4.2.3 过去成功的论证

第[3.6节](https://plato.stanford.edu/entries/reasoning-analogy/#PraBasApp)讨论了斯坦纳（Steiner）认为在物理学中使用“毕达哥拉斯式”类比“引发或应该引发困惑”的观点（1989年：454）。Liston（2000）提供了一个可能的回应：物理学家有权利基于他们过去的成功来使用毕达哥拉斯式的类比。

【科学家】可以承认没有人知道【毕达哥拉斯】推理的原理，并辩称过去类似的策略取得了良好的效果，这已经足够的理由来继续追求，希望在当前情况下取得成功。 （200）

暂且不提关于成功论证的熟悉担忧，真正的问题在于确定什么算作类似的策略。本质上，这就是确定成功的毕达哥拉斯类比的特征。 正如我们所见（§2.4），迄今为止还没有人提供令人满意的方案来描述成功的类比论证，更不用说成功的毕达哥拉斯类比论证了。

### 4.3 *先验*的合理性

一个*先验*的方法追溯了类比推理的模式的有效性，或者特定的类比论证，到一些广泛而基本的原则。这里将概述三种这样的方法。

第一个原因是由凯恩斯（1921）提出的。凯恩斯引用了他著名的独立变量限制原则，他如下所述：

 （LIV）

宇宙中的多样性是有限的，以至于没有一个对象是如此复杂，以至于它的特性可以分为无限数量的独立群体（即，这些群体可以独立存在，也可以同时存在）（1921年：258）。

凭借这个原则和一些额外的假设，凯恩斯能够证明，在没有负类比的情况下，对正类比的了解会增加（逻辑）结论的概率。然而，如果存在一个非平凡的负类比，那么结论的概率将保持不变，正如赫斯（1966年）所指出的。熟悉卡尔纳普的逻辑概率理论的人会认识到，在建立他的框架时，凯恩斯选择了一种不允许从经验中学习的度量方式。

赫尔米亚沿着卡尔纳普的思路对凯恩斯的策略进行了改进。在她的著作《赫尔米亚的阿莫尼俄斯》（1974）中，她提出了她所称之为“聚类假设”的假设：我们的认识概率函数对概括具有内在偏见。对于这种均匀性假设的反对意见是众所周知的（参见萨尔蒙1967年的著作），但即使我们放弃这些反对意见，她的论证也是错误的。这里的主要反对意见——这也适用于凯恩斯——是纯粹的句法公理，如“聚类假设”，无法区分好的类比论证和明显没有价值的类比论证（根据赫尔米亚自己的材料标准，例如）。

巴尔塔（2010年）提出了一种不同的先验策略，将证明的范围限制在满足“良好”类比推理的暂定标准的类比论证上。这些标准是由表达模型（§3.5）指定的。简化形式上，它们要求存在非平凡的正面类比和没有已知的关键反类比。巴尔塔的论证范围也仅限于旨在建立“表面上合理性”而不是概率程度的类比论证。

Bartha的论证基于van Fraassen（1989: 236）提出的对称推理原则：“本质上相同的问题必须得到本质上相同的解决方案。”这个原则的模态扩展大致如下：如果问题“可能是”本质上相同的，那么它们“可能有”本质上相同的解决方案。这里涉及到两种模态。Bartha认为满足表达模型的标准足以建立前提中的模态，即源领域和目标领域在相关方面“可能是本质上相同的”。他进一步提出，从表面上看合理性提供了对后件中的模态的合理解读，即两个领域中的问题“可能有本质上相同的解决方案”。将一个假设称为“从表面上看合理”是将其提升到值得调查的程度，因为它可能是正确的。

这个论证面临两种问题。首先，对对称原则的解释存在疑问。其次，仍然存在一个担忧，即这种证明，像其他所有证明一样，证明了太多。表达模型可能过于模糊或过于宽容。

### 4.4 实用主义的合理化

可以说，对于类比推理最有前景的辩护可能在于其在案例法中的应用（参见[法律推理中的先例和类比](https://plato.stanford.edu/entries/legal-reas-prec/#JusForPre)）。司法决策基于已经支配相关类似案例的裁决和推理，根据*stare decisis*原则（Levi 1949; Llewellyn 1960; Cross and Harris 1991; Sunstein 1993）。法院的个别决定对该法院和下级法院具有*约束力*；法官有义务以“相同的方式”来决定未来的案件。也就是说，个别决定中应用的推理，被称为*ratio decidendi*，必须适用于类似的未来案件（参见[*Example 10*](https://plato.stanford.edu/entries/reasoning-analogy/#Example10)）。当然，在实践中，情况非常复杂。没有两个案例是完全相同的。*ratio*必须在原始案例的事实背景下加以理解，并且对于其普遍性和适用性存在相当大的争议。如果形成共识认为过去的案例判决错误，后续的判决将会将其与新案例区分开来，有效地限制了*ratio*的适用范围仅限于原始案例。

遵循先例的做法可以通过两个主要的实际考虑来证明。首先，最重要的是，这种做法是保守的：它为可复制的决策提供了相对稳定的基础。人们需要能够预测法院的行动并相应地制定计划。判例法作为对任意司法决定的一种制约。其次，这种做法仍然是相当进步的：它允许法律逐渐演变。审慎的法官会区分错误的决定；新的价值观和新的共识可以在一系列决定中逐渐形成。

理论上，判例法在保守和进步社会价值之间取得了健康的平衡。这种证明是务实的。它预设了一套共同的社会价值观，并将类比推理的使用与最优促进这些价值联系起来。还要注意，证明发生在整个实践层面上；个别的类比论证有时会偏离轨道。对判例法的性质和基础的全面审查超出了本条目的范围，但值得问一下：是否可能概括判例法的证明？类比论证是否可以有一个类似的务实证明？

Bartha（2010）试图通过从社会价值观转向认识论价值观来提供这样的正当性。总体思想是，类比推理特别适合实现一套共同的认识论目标或价值观。简单来说，当类比推理符合一定的标准时，它在稳定性和创新需求之间实现了出色（或许是最佳）的平衡。它支持保守的认识论价值观，如简洁性和与现有信念的一致性，以及进步的认识论价值观，如富有成果性和理论统一性（McMullin（1993）提供了一个经典的清单）。

## 5. 超越类比论证

正如前面强调的那样，“类比推理”远不止于类比论证。在本节中，我们将探讨两个广泛的背景，其中类比推理非常重要。

第一个与类比论证密切相关的背景是科学假设的确认。确认是指科学假设根据证据获得归纳支持的过程（参见[evidence](https://plato.stanford.edu/entries/evidence/)、[confirmation](https://plato.stanford.edu/entries/confirmation/)和[Bayes' Theorem](https://plato.stanford.edu/entries/bayes-theorem/)）。确认还可以表示假设H与表达相关证据的命题E之间获得的归纳支持的“逻辑关系”。类比论证能在这个过程中或逻辑关系中发挥作用吗？可以说是可以（对于两者），但这个作用必须仔细界定，并且还有几个障碍需要解决。

第二个背景是前沿科学研究中的概念和理论发展。类比被用来暗示理论概念和思想的可能延伸。推理与合理性考虑相关，但没有直接的类比论证分析。

### 5.1 类比和确认

类比推理与科学假设的确认有何关联？早期章节中的例子和哲学讨论表明，一个好的类比论证确实可以为假设提供支持。但是，有充分的理由怀疑类比提供实际确认的主张。

首先，存在一个逻辑上的困难。为了理解这一点，让我们将确认作为命题之间的关系集中讨论。克里斯滕森（1999: 441）提供了一个有用的一般描述：

有些命题似乎有助于使我们相信其他命题。当我们对E的当前信心有助于使我们对H的当前信心合理时，我们说E证实了H。

在贝叶斯模型中，“信心”以主观概率的形式表示。贝叶斯代理人从对一类命题的主观概率分配开始。确认被理解为一个三元关系：

(11)

*贝叶斯确认* E 相对于 K 确认 H ↔ Pr(H∣E⋅K)>Pr(H∣K)。

E 代表关于已接受证据的命题，H 代表假设，K 代表背景知识，Pr 代表代理人的主观概率函数。确认 H 是指相对于 K 提高其条件概率。从*先验概率* Pr(H∣K) 到*后验概率* Pr(H∣E⋅K) 的转变被称为对 E 的*条件化*。这两个概率之间的关系通常由贝叶斯定理给出（暂不考虑更复杂的条件化形式）。

（12）Pr（H∣E⋅K）=Pr（H∣K）Pr（E∣H⋅K）Pr（E∣K）

对于贝叶斯主义者来说，这里存在逻辑上的困难：类比论证似乎无法提供确认。首先，我们不清楚是否可以将类比论证中包含的信息封装成一个命题E。其次，即使我们能够形式化一个表达该信息的命题E，通常也不适合将其视为证据，因为E中包含的信息已经是背景K的一部分。这意味着E⋅K等同于K，因此Pr（H∣E⋅K）=Pr（H∣K）。根据贝叶斯定义，我们没有确认。 （这是旧证据问题的一个版本，请参见[confirmation](https://plato.stanford.edu/entries/confirmation/)。）第三，也许最重要的是，类比论证通常适用于先验概率Pr（H∣K）甚至未定义的新颖假设H。再次，以贝叶斯条件化的方式定义确认似乎不适用。

如果类比不能通过普通的条件化提供归纳支持，那么是否有其他选择？在这里，我们面临第二个困难，再次在贝叶斯框架内最容易陈述。范弗拉森（1989）对除条件化之外的任何信念更新规则都有一个众所周知的反对意见。这个反对意见适用于任何允许我们在没有新证据的情况下提高置信度的规则。这个批评通过贝叶斯彼得的故事生动地表达了出来，即这些“扩大性”规则容易受到荷兰博弈的攻击。采用任何这样的规则都会导致我们承认一种预见到必然损失的赌注系统是公平的。这类类比推理的规则似乎容易受到范弗拉森的反对意见的攻击。

在贝叶斯认识论中，似乎有至少三种避免这些困难并在其中找到类比论证在其中的角色的途径。首先，有一种我们可以称之为“最小贝叶斯主义”的方法。在贝叶斯框架内，一些作者（Jeffreys 1973；Salmon 1967, 1990；Shimony 1970）认为，“严肃提出的”假设必须具有足够高的先验概率，以使其成为观察结果的首选。Salmon提出类比推理是在这个意义上显示一个假设“严肃”的最重要手段之一。如果类比推理主要针对“先验”概率分配，它可以提供归纳支持，同时在形式上与证实保持明确区分，避免了上述的逻辑困难。这种方法是最小贝叶斯主义，因为它只是提供了一个进入贝叶斯装置的入口，并且只适用于新颖的假设。像德·芬内蒂（de Finetti）（de Finetti和Savage 1972，de Finetti 1974）这样的正统贝叶斯主义者可能不会对类比发挥这种角色有任何问题。

第二种方法是“自由贝叶斯主义”：我们可以以“非规则化的方式”改变我们的先验概率。如果类比论证被认为可以在没有任何新证据的情况下改变对已有假设的观点，那么需要类似这样的方法。这在考古学等领域很常见，作为一种被Wylie称为“将旧数据作为新证据动员起来”的策略（Wylie和Chapman 2016: 95）。正如Hawthorne（2012）所指出的，一些贝叶斯主义者简单地接受了先验概率的初始分配和持续修订（基于合理性论证）都是合理的，但是

贝叶斯归纳的“逻辑”（如此处所述）对于假设的先验可信度评估应该具有什么值没有任何要说的，也不对它们如何改变施加任何限制。

换句话说，通过不规定这种概率修正的任何规则，我们避免了范·弗拉森所指出的困难。这种方法将类比推理纳入贝叶斯帐篷，但承认帐篷的一个黑暗角落，在这个角落中，理性在没有明确规则的情况下运作。

最近，第三种方法引起了人们的兴趣：*类比确认*或*通过类比模拟进行确认*。正如（Dardashti等人，2017）所描述的那样，其思想如下：

我们的核心思想是，在特定情况下，关于无法观测到的现象的预测可以通过在不同系统中进行类比模拟来确认。（57）

达尔达什蒂和他的合著者专注于一个特定的例子（***例子17***）：“哑洞”和其他类比引力黑洞（Unruh 1981; Unruh 2008）。与真实的黑洞不同，其中一些类比可以（事实上已经）在实验室中实施和研究。鉴于我们对这些系统的模型与我们对黑洞的模型之间的确切形式类比，以及某些重要的附加假设，达尔达什蒂等人提出了一个有争议的观点，即关于类比体系的观察结果提供了关于实际黑洞存在的证据。例如，在类比系统中观察到类似于霍金辐射的现象将为黑洞中存在霍金辐射提供确认。在第二篇论文中（Dardashti et al. 2018, 其他互联网资源），他们在贝叶斯框架内发展了确认的案例。

明确阐述类比确认机制的吸引力是显而易见的。它将为探索宇宙学以及考古学、进化生物学等历史科学领域中无法触及的现象的确认提供工具，并且还将在医学科学的某些领域中发挥作用，因为伦理约束规定了不能对人类进行实验。此外，正如达达什蒂等人所指出的，类比确认依赖于从类比系统中获得的*新证据*，因此不容易受到上述逻辑困难的影响。

尽管类比确认的概念并非全新（可以考虑动物实验，如[*例子8*](https://plato.stanford.edu/entries/reasoning-analogy/#Example8)所述），但达达什蒂等人（2017年，2018年[其他互联网资源]）的主张需要评估。对于黑洞的例子，一个直接的困难是：如果我们从普通的类比论证角度来思考，那么就*没有积极的类比*，因为简单来说，我们没有关于“哑洞”和黑洞之间已知相似性的基础。正如克劳瑟等人（2018年，其他互联网资源）所争论的，“我们不知道霍金辐射推导中使用的特定建模框架是否*实际上描述了黑洞*。”这可能不会让达达什蒂等人感到担忧，因为他们声称类比确认与普通的类比论证是不同的。类比确认可能在动物实验等存在已知相似性基础的情况下与我们只能通过理论模型来接触目标领域的情况下有所不同。

### 5.2 概念变化与理论发展

在[§3.6](https://plato.stanford.edu/entries/reasoning-analogy/#PraBasApp)中，我们看到基于实践的类比研究为评估类比论证提供了洞察力。这些研究还指出了类比的*动态*或*规划*角色，这似乎需要超越为类比论证所开发的评估框架。

Knuttila和Loettgers（2014）研究了类比推理在合成生物学中的作用，这是一个跨学科领域，涉及物理学、化学、生物学、工程学和计算科学。在这个领域中，类比的主要作用不是构建个别的类比论证，而是发展诸如“噪声”和“反馈环路”等概念。这些概念不断完善，通过对工程和物理系统中类似概念的正面和负面类比的引导。在这里，类比推理是“短暂的、异质的和有计划的”（87）。负面类比在个别类比论证的焦点是理论构建和概念完善时，扮演着突出和建设性的角色。

类比推理在应用于另一个前沿领域——新兴引力学中也有类似的观察。在这个物理学领域中，不同的理论方法描绘了引力从不同的微观结构中产生（Linneman和Visser 2018）。在引力理论中出现了在微观层面上不存在的“新颖而稳健”的特征。利用与其他新兴现象（如流体力学和热力学）的类比来塑造这些提议。与合成生物学类似，类比推理的主要目标不是针对个别论证的制定和评估，而是发展不同的引力理论模型。

这些研究探索了类比在科学研究前沿的流动和创造性应用，充分的分析肯定会超越我们关注的个别类比论证的分析。Knuttila和Loettgers（2014）认为个别类比论证不是类比推理的“主要单位”，但这是一个有争议的结论。例如，Linneman和Visser（2018）明确肯定了通过“典型类比论证”评估不同引力模型的重要性：

> 我们接受了提出新兴引力范式的明确论证的挑战... 论证只能是启发式层面上的合理性论证，并不意味着它们免于审查和全面的批判评估。物理学哲学家在量子引力发现过程中的工作... 应该正是提供这种准确评估的工作。（Linneman和Visser 2018：12）

据此，林内曼和维瑟为每个新兴引力模型提出明确的类比论证，并使用评估个别类比论证的熟悉标准对其进行评估。可以说，即使是最雄心勃勃的启发式目标仍然依赖于考虑合理性的因素，这些因素通过类比论证的方式表达和审查，从而获益。
