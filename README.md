## 别名词条

- 凡是以 `*` 开头的词条都是指向其他词条的“别名词条”，实际上是不存在的，例如 `*弗朗索瓦-马里‧阿鲁埃——见伏尔泰 Arouet, François-Marie — see Voltaire`，伏尔泰（Voltaire）是其笔名，“弗朗索瓦-马里‧阿鲁埃（François-Marie Arouet）”才是原名，但是实际上两个代表的是一个人。
- `*` 只代表当前的词条，不代表下辖的所有条目，凡是别名词条都会一一标注。例如 `*美学 aesthetics` 不代表美学下面的所有词条都是别名词条，只有有 `*` 标注的才是别名词条。
